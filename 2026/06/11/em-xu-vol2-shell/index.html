第二十章 壳体的一般理论

§20-1 壳体的中面几何

壳体：厚度 $h$ 远小于曲率半径与中面尺寸；中面为曲面。

正交曲线坐标 $(\alpha, \beta)$，$z$ 沿法向。

曲率与扭率

主曲率 $k_1, k_2$，曲率半径 $R_1, R_2$。

高斯曲率 $K = k_1 k_2$。

§20-2 壳体的内力与变形

单位长度内力：

  • 膜力 $N_\alpha, N_\beta, N_{\alpha\beta}$（中面内）；
  • 弯矩 $M_\alpha, M_\beta, M_{\alpha\beta}$；
  • 横剪力 $Q_\alpha, Q_\beta$。

中面应变 $\varepsilon_\alpha, \varepsilon_\beta, \gamma_{\alpha\beta}$；曲率变化 $\kappa_\alpha, \kappa_\beta, \kappa_{\alpha\beta}$。

§20-3 壳体的平衡微分方程

（无体力，一般形式较繁；见教材。）

膜力平衡 + 弯矩平衡 + 剪力与弯矩关系（忽略 $Q$ 对弯矩贡献时为无矩理论）。

§20-4 壳体的物理方程

广义胡克定律联系内力与中面变形、曲率变化；厚度积分得 $N$-$ \varepsilon$、$M$-$\kappa$ 关系，含 $D = Eh^3/[12(1-\nu^2)]$。

§20-5 壳体的边界条件

固定、简支、自由边；8 个边界条件（取决于理论阶数）。

§20-6 薄膜理论（无弯矩理论）

弯曲刚度忽略，仅膜力平衡：

$$\frac{\partial N_\alpha}{\partial \alpha} + \frac{\partial N_{\alpha\beta}}{\partial \beta} + \cdots + q_\alpha = 0$$

适用于光滑壳体、法向荷载；边界附近、集中荷载处需弯曲修正。

薄膜与弯曲理论何时用

情形 理论
球壳内压、圆柱内压（远离边） 薄膜
壳端封头、开孔、支座 边缘弯曲
集中荷载 局部弯曲 + 薄膜

第二十一章 柱壳

§21-1 柱壳的几何与荷载

中面为圆柱面 $r = R$，坐标 $(x, \theta)$ 或 $(\xi, \theta)$。

典型荷载：自重、内压、风载。

§21-2 轴对称问题

仅与 $x$ 有关，$N_\theta$ 平衡内压 $p$：

$$N_\theta = pR \quad \text{（内压筒壳）}$$

环向薄膜应力 $\sigma_\theta = pR/h$。

薄膜平衡推导

取单位长度柱壳条带，环向内力 $N_\theta$（单位长度上的力），内压合力 $p \cdot 2R$（单位长度），平衡 $2N_\theta = p \cdot 2R$，故 $N_\theta = pR$。膜应力 $\sigma_\theta = N_\theta/h = pR/h$。两端封闭时，轴向平衡给出 $\sigma_x = pR/(2h)$。

§21-3 非轴对称问题

将荷载、内力展为 $\theta$ 向傅里叶级数，化为常微分方程。

§21-4 边缘效应

端部约束或荷载突变引起局部弯曲，影响长度 $\lambda = \sqrt[4]{R^2 h^2 / [12(1-\nu^2)]}$。

第二十二章 旋转壳

§22-1 旋转曲面几何

母线绕轴旋转得球壳、锥壳、椭球壳等；主曲率 $R_1, R_2$。

§22-2 球壳

内压 $p$：薄膜理论

$$\sigma_\theta = \sigma_\varphi = \frac{pR}{2h}$$

（均匀拉伸，无弯矩。）

§22-3 锥壳

薄膜力与弯矩联合；顶点、底边边界条件重要。

§22-4 容器设计应用

压力容器、储罐球顶、冷却塔双曲壳等工程简化计算。

第二十三章 扁壳

§23-1 扁壳假设

高斯曲率小，中面可展成 $z = f(x,y)$，$|f| \ll$ 跨度；曲率半径大。

§23-2 扁壳方程

化为两个四阶偏微分方程（或一个八阶方程），联系 $w$ 与膜力函数 $F$：

$$D\nabla^4 w + \nabla^2 F = q$$

$$\nabla^4 F + Eh\left(\frac{\partial^2 w}{\partial x^2}\frac{\partial^2 w}{\partial y^2} - \left(\frac{\partial^2 w}{\partial x \partial y}\right)^2\right) = 0$$

（大挠度扁壳含非线性项。）

§23-3 矩形底扁壳

四边简支、嵌固等边界，Navier 型级数解。

§23-4 应用

屋盖、展览馆大跨度结构。
