主板选购：芯片组、命名与平台搭配

发表于2026-06-05
|浏览量:

主板连接 CPU、内存、显卡与存储，决定 平台代数、扩展接口与供电上限。须与 CPU 选购：品牌、命名与性能分级 所选处理器 插槽与芯片组 一致；与 显卡选购：品牌、命名与性能分级、后续内存与硬盘型号也需核对接口。下文介绍主要板厂系列、Intel / AMD 芯片组命名及 2025–2026 年大陆电商常见价位（仅供参考）。

型号中常含 芯片组字母（如 B760M、X670E）与 板型（M = Micro-ATX，I / ITX = Mini-ITX，无后缀多为 ATX）。

Intel 平台与芯片组

插槽代际（与 CPU 对应）

插槽 常见 CPU 备注
LGA1700 12 / 13 / 14 代酷睿 DDR4 或 DDR5 主板并存，买前确认
LGA1851 15 代酷睿（Arrow Lake 等） DDR5 only；与 LGA1700 不兼容

芯片组命名（桌面）

芯片组 定位 典型能力
H610 / H710 入门 扩展少，适合 i3 / 非 K 的 i5
B760 / B860 主流 性价比最高；可跑 K 系 CPU，超频能力有限或依代际
Z790 / Z890 高端 CPU 超频、更多 PCIe / M.2、更强供电

字母 B / H 偏性价比；Z 面向超频与多扩展。同代 Z > B > H。

AMD 平台与芯片组

插槽

插槽 常见 CPU 备注
AM5 Ryzen 7000 / 9000 系 DDR5 only；长周期平台
AM4 Ryzen 5000 及更早 DDR4；新装机已少见，多为存量升级

芯片组命名（AM5 桌面）

芯片组 定位 典型能力
A620 入门 扩展精简，适合 Ryzen 5 非 X
B650 / B650E 主流 绝大多数装机首选；E 表示部分 PCIe 5.0 通道
X670 / X670E 高端 更多 M.2 与 PCIe；X670E 强调 PCIe 5.0
B850 / X870(E) 9000 系新板 与 Ryzen 9000 同步迭代，规格类比上代 B650 / X670

B650 与 X670 均可搭配 Ryzen 7 / 9；差异在扩展与供电余量，非「能否点亮」。

板型与型号后缀

板型 尺寸（约） 场景
ATX 305 × 244 mm 标准大板，扩展最多
Micro-ATX（M） 244 × 244 mm 小机箱、性价比
Mini-ITX 170 × 170 mm 紧凑主机，接口最少、价高

常见后缀：-WIFI / AX 表示板载无线；DDR4 / DDR5 标注内存类型（Intel 老平台）；II 多为小改版。

性能分级与用途

档位 典型组合 适合
入门 H610 / A620 + mATX 办公、核显机、Ryzen 5 非 X
主流 B760 / B650 + mATX 或 ATX 游戏与日常使用；i5 / Ryzen 5–7
中高端 B760 高配 / B650E、X670 多 M.2、2K–4K 游戏、轻度创作
旗舰 Z790 / Z890 / X670E K / X 系超频、多硬盘、重度负载

供电相数与散热马甲影响 高功耗 CPU 长时间满载；i9 / Ryzen 9 建议 12+ 相级以上 主流板，而非最低配 H / A 系。

大致价格区间

大陆电商常见区间（元，约 2025 下–2026 初，同芯片组华硕 ROG 高于入门款）：

档位 Intel（举例） AMD AM5（举例） 大致价位
入门 H610 DDR4 mATX A620 mATX ¥400–650
主流 B760 DDR5 mATX B650 mATX ¥700–1,100
主流 ATX B760 / B860 ATX B650 ATX ¥900–1,400
中高端 B760 高配 / Z790 入门 B650E / X670 入门 ¥1,200–1,800
旗舰 Z790 / Z890 高端 X670E / X870E ¥2,000–4,500+
ITX 溢价 任意芯片组 ITX AM5 ITX 同芯片组 +¥300–1,000
  • DDR4 版 B760 因存量常略便宜；新装机更推荐 DDR5 + AM5 或 Intel 新平台。
  • 板载 WiFi 版通常贵 ¥100–200。

选购要点

  1. CPU 与插槽第一：LGA1700 ≠ LGA1851；AM4 ≠ AM5。与 CPU 选购：品牌、命名与性能分级 一并确定。
  2. 内存类型：主板标注 DDR4 或 DDR5，不可混插。
  3. 显卡与硬盘：确认 PCIe x16 显卡槽 数量、M.2 接口 数量与 PCIe 版本；NVMe 见 存储选购：固态、机械硬盘与接口。
  4. 内存：DDR4 或 DDR5 与容量、频率见 内存选购：DDR4 / DDR5、品牌与频率。
  5. BIOS 与代际：老主板刷 BIOS 支持新 CPU 需查官网 CPU 支持列表（如 AM5 上 9000 系）。
  6. 后置 I/O：USB-A / C 数量、2.5G 网口、HDMI（核显用户）是否够用。
  7. 机箱兼容：ATX 板需 ATX 机箱；ITX 需 SFX 电源与小机箱，见 机箱选购：规格、风道与兼容性。
