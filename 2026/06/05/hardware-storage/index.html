存储选购：固态、机械硬盘与接口

发表于2026-06-05
|浏览量:

存储负责系统、软件与数据的持久保存。NVMe 固态（SSD） 为新装机系统盘首选；机械硬盘（HDD） 仍适合大容量冷数据备份。须与 主板选购：芯片组、命名与平台搭配 的 M.2 / SATA 接口匹配。下文介绍主要品牌、命名规则、性能分级与 2025–2026 年大陆电商常见价位（仅供参考）。

固态与机械

类型 优点 缺点 典型用途
NVMe SSD 速度快、无噪音、抗震 单位容量价高；高端需散热 系统盘、游戏盘、创作素材
SATA SSD 价格低、兼容老平台 上限约 550 MB/s 老主板升级、副盘
HDD 每 GB 最便宜 慢、噪音、怕震动 影音库、备份、监控录像

新装机建议：至少一块 NVMe 作系统盘；大容量需求可加 HDD 或第二块 SSD。

机械硬盘看 转速（5400 / 7200 RPM）与 缓存（64–256 MB）；NAS 盘强调 7×24 运行 与 CMR（传统磁记录，避免 SMR 作系统盘）。

NAND 颗粒：TLC 与 QLC 详解

固态硬盘的数据写在 NAND 闪存颗粒 上。颗粒按每个存储单元存放的 比特数 分级，比特越多，单颗容量越大、成本越低，但 耐久与持续写入性能 通常越弱。消费级 NVMe 与 SATA SSD 以 TLC、QLC 为主；更高阶的 SLC / MLC 多见于企业盘或颗粒内部的模拟缓存区。

分级一览

类型 每单元比特 俗称 消费级可见度 概括
SLC 1 bit 单层单元 极少（多为缓存模拟） 最快、最耐用；单位容量最贵
MLC 2 bit 多层单元 存量 / 企业 性能与寿命介于 SLC 与 TLC
TLC 3 bit 三层单元 主流 性能、价格、寿命均衡
QLC 4 bit 四层单元 大容量入门盘 便宜、容量大；重写入与缓存耗尽后变慢
PLC 5 bit 五层单元 试点 / 企业向 进一步压成本；消费 DIY 少见

下文重点说明装机最常见的 TLC 与 QLC。

工作原理（简化）

写入数据时，控制器对 NAND 单元施加不同电压，表示 0 / 1 组合。单元状态越多（TLC 8 种、QLC 16 种），区分电压的裕量越小，导致：

  • 编程（写入）更慢，需更精细校验；
  • 擦除次数（P/E 循环）有限，寿命随写入量累积而下降；
  • 单元间 干扰（crosstalk） 更明显，长期或高温下误码率上升，依赖控制器纠错（ECC）。

因此不能只看接口 PCIe 4.0 / 5.0，颗粒类型同样影响 长期使用体验。

TLC（Triple-Level Cell）

特点

维度 说明
容量成本 低于早期 MLC，高于 QLC；1 TB / 2 TB 主流盘多为 TLC
性能 顺序与随机性能均衡；持续写入在缓存外仍较稳定
耐久 标称 TBW 通常高于同容量 QLC（如 1 TB 盘 600–1200 TBW 级，视型号而定）
适用 系统盘、游戏盘、系统 + 常用软件、轻度剪辑缓存

常见场景

  • 三星 990 PRO、西数 SN850X、致态 TiPlus7100 等 性能向型号 多为 TLC（具体以厂商规格 / 评测为准）。
  • 无 DRAM 的 TLC 盘（DRAM-less）仍可作为系统盘，大文件连续写入时需关注评测中的 缓外速度。

QLC（Quad-Level Cell）

特点

维度 说明
容量成本 同晶圆可做出更大容量或更低售价；4 TB / 8 TB 入门 NVMe 常见 QLC
性能 依赖 SLC 缓存（见下节）；缓存命中时接近 TLC；缓外 可降至数百 MB/s 甚至更低
耐久 P/E 与 TBW 通常 低于 TLC；不适合长期高强度写入
适用 游戏库（读多写少）、影音仓库、副盘、从 HDD 升级的大容量盘

常见场景

  • 西数 SN580、部分「大容量蓝盘」、低价 PCIe 4.0 白牌：多为 QLC。
  • 将 QLC 用作 唯一系统盘 可行，但应避免：大量下载写入、视频剪辑输出目录、虚拟机磁盘高 IO。

SLC 缓存（为何 QLC 标称速度仍很高）

厂商在部分容量区域用 SLC 模式（每单元只存 1 bit）做写入缓冲，称 SLC Cache / pSLC：

概念 含义
缓内 写入落在 SLC 缓存内，接近宣传峰值
缓外 缓存耗尽后，速度取决于颗粒本体（QLC 缓外常为数 MB/s–1 GB/s 级）
恢复 空闲一段时间后，控制器将缓存数据搬回主存储，缓存空间恢复

家用影响：日常开机、浏览、游戏加载以 读 为主，多数时间感觉不到缓外；一次性复制上百 GB、安装大型游戏连续写入时，QLC 可能明显变慢。

TLC 与 QLC 对比表

对比项 TLC QLC
单位容量价格 较高 更低
标称顺序速度 高（同样受缓存影响） 标称相近，缓外差距大
随机 4K / 系统响应 通常更好 可用；重载写入时波动大
TBW / 寿命 较高 较低
发热 中等 持续写入时可能更明显
推荐角色 系统盘、主力工作盘 副盘、游戏 / 媒体库
识别方式 评测拆解、官网规格、型号世代 同上；过分低价大容量需警惕

耐久：TBW 与日常是否够用

TBW（Total Bytes Written）表示厂商建议的保修期内写入总量。粗算：

QLC 盘 TBW 更低（例如 1 TB 盘 200–400 TBW 级），对 普通家用 仍往往够用；风险在于 长期视频剪辑、挖矿、P2P 疯狂下载 等持续写入。若工作流如此，应选 TLC 高 TBW 或企业级盘。

选购建议（结合颗粒）

用途 建议颗粒 说明
系统 + 办公 TLC 稳定、缓外可接受
3A 游戏主力盘 TLC 或口碑 QLC 安装阶段写入大，之后以读为主；QLC 可省预算
2 TB+ 游戏 / 电影库 QLC 可接受 读多写少，注意预留空间
视频剪辑输出 TLC 旗舰 / 高 TBW 避免 QLC 作写入盘
虚拟机 / 数据库 / NAS 缓存 TLC 及以上 持续随机写入多

如何确认颗粒类型：包装盒与商品页常不标明；可查 厂商白皮书、TechPowerUp / 快科技等评测拆解，或选购标明 TLC 的型号（如部分系列直接写 3D TLC）。

接口：NVMe 与 SATA

接口 物理形态 带宽（约） 主板连接
M.2 NVMe M.2 键位 M PCIe 4.0 x4 ≈ 7 GB/s 主板 M.2 槽
M.2 SATA M.2 键位 B+M ≈ 550 MB/s 部分老 M.2 槽
SATA 2.5” 数据线 + 供电 ≈ 550 MB/s SATA 口
SATA 3.5” HDD 数据线 + 供电 100–200 MB/s 级 SATA 口

注意：

  • 主板 M.2 槽可能与其它 SATA 口 共享通道，插满需查说明书。
  • PCIe 5.0 NVMe 需主板与 CPU 支持，且 建议散热片，否则易降速。
  • 老平台无 M.2 可用 SATA SSD + 2.5 寸位 或 PCIe 转接卡（兼容性视主板而定）。

性能分级与用途

档位 典型产品（举例） 适合
入门 NVMe SN580、NV3、TiPlus7100 系统 + 办公；PCIe 4.0 无缓存放量
主流 NVMe SN770、KC3000、TiPlus7100 高配 游戏、日常创作
高端 NVMe 990 PRO、SN850X、PCIe 5.0 旗舰 4K 素材、大文件频繁读写
SATA SSD 870 EVO、致态 SC001 等 老机升级、副盘
HDD 消费 酷鱼 2–4 TB 游戏库、电影
HDD NAS 红盘 / 狼盘系列 家用 NAS、7×24

容量建议

场景 系统盘 扩展
办公 500 GB–1 TB NVMe —
游戏 1 TB NVMe 可选第二块 2 TB 游戏 / HDD
剪辑 1 TB 系统 + 2 TB 素材 NVMe HDD 归档
大仓库 1 TB 系统 4–8 TB HDD 或 NAS

类型 容量 大致价位

PCIe 4.0 NVMe 旗舰 1 TB ¥750–1,100

大促期间 1 TB PCIe 4.0 主流盘常降至 ¥400 左右；PCIe 5.0 与旗舰溢价明显。

选购要点

  1. 系统盘优先 NVMe：与 主板选购：芯片组、命名与平台搭配 M.2 槽数量、PCIe 版本核对。
  2. 不必盲目 PCIe 5.0：Gen4 旗舰对多数用户已过剩；Gen5 需散热与平台支持。
  3. TLC vs QLC：见 [NAND 颗粒：TLC 与 QLC 详解](#NAND 颗粒：TLC 与 QLC 详解)；勿仅凭标称 MB/s 判断。
  4. 分区与备份：系统盘预留 15%–20% 空余；重要数据 3-2-1 备份，勿只信一块盘。
  5. HDD 勿当系统盘：新装机仅预算极紧时考虑 SATA SSD 起步。
  6. 马甲与散热：主板 M.2 散热片可复用；发热大旗舰建议加装。
