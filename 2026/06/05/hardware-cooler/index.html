散热器选购：风冷、水冷与机箱风扇

发表于2026-06-05
|浏览量:

CPU 散热器将处理器热量导出至机箱风道；机箱风扇则强化整机换气。选型须满足 CPU 选购：品牌、命名与性能分级 功耗、机箱选购：规格、风道与兼容性 散热器限高 与主板周围空间，并与 电源选购：功率、认证与模组 风道布局配合。下文介绍主要品牌、风冷 / 水冷分类、性能分级及 2025–2026 年大陆电商常见价位（仅供参考）。

机箱风扇（可混用品牌）

品牌 说明
猫头鹰 静音首选
利民、九州风神 高性价比 PWM
联力、海盗船 与机箱套装或 RGB 生态

风冷与水冷

类型 结构 优点 缺点
单塔风冷 单组鳍片 + 1–2 风扇 便宜、维护简单 极限 TDP 较低
双塔风冷 双鳍片堆叠 接近 240 水冷性能；无漏液风险 体积大、挡内存
一体式水冷 AIO 冷排 + 水泵 + 冷头 外观整洁、利于机箱风道 价高；泵寿命与安装
分体水冷 自定义冷排与管路 极限散热与展示 成本高、维护难，DIY 进阶

办公 / 低功耗 CPU 可用盒装自带散热器；游戏与创作 建议单塔以上或 240 水冷。

命名与关键参数

风冷

示例：Thermalright PA120 SE

参数 含义
塔数 单塔 / 双塔
热管数量 4 / 6 管等；镀镍工艺影响寿命
风扇尺寸 120 mm 最常见
TDP 标称 厂商宣称散热能力（W）；作粗参考
高度 须 < 机箱 CPU 限高

一体水冷（AIO）

示例：DeepCool LT720（360）

参数 含义
冷排尺寸 120 / 240 / 360 / 420（mm，对应 1 / 2 / 3 / 4 风扇位）
冷头 是否挡内存、是否带屏幕
扣具 须支持 LGA1700 / LGA1851 / AM5 等，见包装列表

通用

参数 含义
PWM 4pin 温控风扇，主板 BIOS 调速
ARGB / RGB 灯效，需主板 5V 3pin 或控制器
扣具兼容 买前核对 CPU 插槽

性能分级与 TDP 匹配

CPU 档位 典型 TDP / 功耗 建议散热
办公 i3 / Ryzen 5 非 X 65 W 级 盒装 / 单塔 4 热管
主流 i5 / Ryzen 5 65–105 W 单塔双风扇 / 240 水冷
高端 i7 / Ryzen 7 105–170 W 双塔 / 240–360 水冷
旗舰 i9 / Ryzen 9 / K 系超频 200 W+ 峰值 双塔旗舰 / 360 水冷

以上为风道正常时的经验值；机箱闷罐需 升一档 散热规格。

水冷冷排安装位置：

位置 说明
顶部出风 常见；冷排热风由顶部排出
前置进风 冷排进风可降低 CPU 温，但 GPU 温度可能升高
侧板 部分机箱支持 240/280

机箱风扇

规格 说明
120 / 140 mm 与 机箱选购：规格、风道与兼容性 风扇位一致
PWM 集线器 风扇多而主板 4pin 不足时使用
风道建议 前 / 下进风，后 / 顶出风；与 CPU 散热器风向一致
配置档 风扇数量（参考）
入门 机箱自带 1–2 把即可
游戏 前 2 进 + 后 1 出；或前 3 进 + 顶 2 出
高功耗 满配风扇位 + 监控温度

大陆电商常见区间（元，约 2025 下–2026 初）：

类型 举例 大致价位

选购要点

  1. 限高第一：散热器高度 ≤ 机箱标称；双塔注意 内存马甲高度 与风扇是否抬高。
  2. 扣具与插槽：AM5 / LGA1700 / LGA1851 扣具不同，勿混用旧版。
  3. 内存兼容性：双塔可能挡第一、二内存槽，需查评测或选矮条。
  4. 水冷与机箱：360 冷排需机箱支持；顶部安装注意冷排与主板供电散热片间隙。
  5. 硅脂：多数散热器预涂硅脂；拆装后重涂可用利民、猫头鹰等成品硅脂。
  6. 噪音：BIOS 设置风扇曲线；静音优先选猫头鹰或低转速 PWM 策略。
  7. 整机风道：CPU 散热与机箱风扇一体规划，见 机箱选购：规格、风道与兼容性。
