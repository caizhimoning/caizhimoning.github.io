第十三章 薄板的小挠度弯曲

§13-1 基本概念与假设

薄板：厚度 $h \ll$ 平面尺寸；中面为 $xy$ 平面，$z$ 沿厚度。

小挠度（Kirchhoff 板理论）：挠度 $w \ll h$。

基本假设

  1. 中面法线变形后仍为直线且垂直于中面（忽略横向剪切 $\gamma_{xz}, \gamma_{yz}$）；
  2. 中面法向应力 $\sigma_z$ 相对 $\sigma_x, \sigma_y$ 可忽略（平面应力）；
  3. 挠度 $w$ 远小于板厚。

§13-2 内力矩与曲率关系

曲率与挠度的关系

Kirchhoff 假设：中面法线变形后仍垂直于中面，且长度不变。中面沿 $x$ 曲率等于挠度曲面的曲率，小挠度下

$$\kappa_x = -\frac{\partial^2 w}{\partial x^2}, \quad \kappa_y = -\frac{\partial^2 w}{\partial y^2}, \quad \kappa_{xy} = -\frac{\partial^2 w}{\partial x \partial y}$$

$\kappa_{xy}$ 为扭率（单位弧度）。

弯矩与曲率关系的推导

取厚度方向 $z \in [-h/2, h/2]$。由平面应力假设，$\sigma_x,\sigma_y,\tau_{xy}$ 沿 $z$ 线性分布（纯弯曲）：

$$\sigma_x = \frac{12M_x}{h^3}z, \quad \sigma_y = \frac{12M_y}{h^3}z, \quad \tau_{xy} = \frac{12M_{xy}}{h^3}z$$

代入本构 $\varepsilon_x = \dfrac{1}{E}(\sigma_x - \nu\sigma_y)$，并利用 $\varepsilon_x = z\kappa_x$（距中面 $z$ 处纤维应变）：

$$z\kappa_x = \frac{z}{Eh^3}\bigl(12M_x - 12\nu M_y\bigr) \quad \Rightarrow \quad M_x = \frac{Eh^3}{12(1-\nu^2)}(\kappa_x + \nu\kappa_y)$$

定义 抗弯刚度 $D = \dfrac{Eh^3}{12(1-\nu^2)}$，则

$$M_x = D(\kappa_x + \nu\kappa_y), \quad M_y = D(\kappa_y + \nu\kappa_x), \quad M_{xy} = D(1-\nu)\kappa_{xy}$$

§13-3 平衡微分方程

微元体的力矩与力平衡

取中面微元 $dx,dy$，单位宽度。横向剪力 $Q_x,Q_y$，弯矩 $M_x,M_y,M_{xy}$。对 $y$ 轴列力矩平衡（略高阶项）：

$$\frac{\partial M_x}{\partial x} + \frac{\partial M_{xy}}{\partial y} = Q_x$$

对 $x$ 轴同理：$\dfrac{\partial M_{xy}}{\partial x} + \dfrac{\partial M_y}{\partial y} = Q_y$。

再对 $z$ 向列力平衡（向上为正）：

$$\frac{\partial Q_x}{\partial x} + \frac{\partial Q_y}{\partial y} + q = 0$$

消去剪力：将 $Q_x = \dfrac{\partial M_x}{\partial x}+\dfrac{\partial M_{xy}}{\partial y}$ 代入第三式，

$$\frac{\partial^2 M_x}{\partial x^2} + 2\frac{\partial^2 M_{xy}}{\partial x \partial y} + \frac{\partial^2 M_y}{\partial y^2} + q = 0$$

将 $M_x,M_y,M_{xy}$ 用曲率、再用 $w$ 表示：

$$D\nabla^4 w = D\left(\frac{\partial^4 w}{\partial x^4} + 2\frac{\partial^4 w}{\partial x^2 \partial y^2} + \frac{\partial^4 w}{\partial y^4}\right) = q(x,y)$$

此即薄板弯曲的双调和方程，形式与平面 Airy 方程相同，但物理量 $w$ 为挠度，$q$ 为横向荷载。

§13-4 边界条件

边界 位移条件 力条件
固定边 $w=0$，$\partial w/\partial n = 0$ —
简支边 $w=0$，$M_n=0$ —
自由边 $M_n=0$，$Q_n + \partial M_{ns}/\partial s = 0$（Kirchhoff 有效剪力） —

§13-5 Navier 解（四边简支矩形板）

边长 $a \times b$，均布荷载 $q_0$：

$$w = \sum_{m=1,3,5\ldots}\sum_{n=1,3,5\ldots} \frac{16q_0}{D\pi^6 m n (m^2/a^2 + n^2/b^2)^2} \sin\frac{m\pi x}{a}\sin\frac{n\pi y}{b}$$

级数快速收敛。

Navier 解的求法

四边简支：$x=0,a$ 与 $y=0,b$ 上 $w=0$，$M_n=0$。设

$$w = \sum_{m=1}^{\infty}\sum_{n=1}^{\infty} A_{mn} \sin\frac{m\pi x}{a}\sin\frac{n\pi y}{b}$$

代入 $D\nabla^4 w = q_0$（均布荷载），利用

$$\frac{\partial^4}{\partial x^4}\sin\frac{m\pi x}{a} = \left(\frac{m\pi}{a}\right)^4 \sin\frac{m\pi x}{a}$$

及 $\nabla^4\bigl(\sin\frac{m\pi x}{a}\sin\frac{n\pi y}{b}\bigr) = \left[\left(\frac{m\pi}{a}\right)^2 + \left(\frac{n\pi}{b}\right)^2\right]^2 \sin\frac{m\pi x}{a}\sin\frac{n\pi y}{b}$，比较系数：

$$A_{mn} = \frac{16q_0}{D\pi^6 m n \left(m^2/a^2 + n^2/b^2\right)^2} \quad (m,n \text{ 为奇数})$$

$m,n$ 为偶数时 $A_{mn}=0$（对称荷载下偶数项不出现）。

§13-6 Levy 解（对边简支）

$y=0, b$ 简支，$x=0, a$ 任意（固支、自由等），将 $w$ 展为 $y$ 向正弦级数，化为常微分方程求 $x$ 向解。

§13-7 圆板的轴对称弯曲

极坐标，$w = w(r)$：

$$D\left(\frac{\mathrm{d}^4 w}{\mathrm{d}r^4} + \frac{2}{r}\frac{\mathrm{d}^3 w}{\mathrm{d}r^3} - \frac{1}{r^2}\frac{\mathrm{d}^2 w}{\mathrm{d}r^2} + \frac{1}{r^3}\frac{\mathrm{d}w}{\mathrm{d}r}\right) = q(r)$$

均布荷载圆板，周边简支或嵌固有解析解。

第十四章 矩形薄板的差分解法

对 $\nabla^4 w = q/D$ 列差分方程，内点 13 点格式，边界用弯矩、剪力差分表达，解 $w_{ij}$。

第十五章 用变分法解薄板弯曲

薄板应变能：

$$U = \frac{D}{2}\iint \left[(\nabla^2 w)^2 - 2(1-\nu)\left(\frac{\partial^2 w}{\partial x^2}\frac{\partial^2 w}{\partial y^2} - \left(\frac{\partial^2 w}{\partial x \partial y}\right)^2\right)\right] \mathrm{d}x\mathrm{d}y$$

最小势能 $\Pi = U - \iint qw,\mathrm{d}x\mathrm{d}y$，里兹法设 $w = \sum a_{mn} W_{mn}(x,y)$。

第十六章 薄板的振动

自由振动 $w = W(x,y)\sin(\omega t + \phi)$：

$$D\nabla^4 W = \rho h \omega^2 W$$

$\rho$ 为材料密度；$\omega$ 为固有圆频率。

矩形简支板基频可查表；与 理论力学（Ⅱ）· 机械振动基础 板振动类比。

第十七章 薄板的稳定

面内压力 $N_x, N_y$ 作用下临界屈曲：

矩形板单向压缩临界荷载（简支）：

$$N_{cr} = \frac{\pi^2 D}{b^2}\left(\frac{b}{a} + \frac{a}{b}\right)^2$$

$a, b$ 为边长；长板 $a \gg b$ 时 $N_{cr} \approx \dfrac{\pi^2 D}{b^2}$。

第十八章 各向异性板

正交各向异性板，$x,y$ 为主方向：

$$D_x \frac{\partial^4 w}{\partial x^4} + 2H \frac{\partial^4 w}{\partial x^2 \partial y^2} + D_y \frac{\partial^4 w}{\partial y^4} = q$$

$D_x, D_y, H$ 由 $E_x, E_y, \nu_{xy}, G_{xy}$ 与 $h$ 确定。复合材料层合板常用。

第十九章 薄板的大挠度弯曲

$w$ 与 $h$ 同阶时，中面膜应变不可忽略，平衡方程含 $w$ 的非线性项：

$$D\nabla^4 w - \frac{h}{2}\frac{E}{1-\nu^2}\left[\left(\frac{\partial^2 w}{\partial x^2}\right)^2 + \left(\frac{\partial^2 w}{\partial y^2}\right)^2 + 2\left(\frac{\partial^2 w}{\partial x \partial y}\right)^2\right] = q$$

膜力与弯曲耦合；圆板大挠度有 Föppl 解。
