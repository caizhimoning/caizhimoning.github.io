电源选购：功率、认证与模组

发表于2026-06-05
|浏览量:

电源（PSU）为整机供电，质量直接影响稳定性与硬件寿命。功率须覆盖 CPU 选购：品牌、命名与性能分级、显卡选购：品牌、命名与性能分级 及余量；接口须满足显卡供电与 存储选购：固态、机械硬盘与接口、风扇等扩展。下文介绍主要品牌、命名与认证、功率分级及 2025–2026 年大陆电商常见价位（仅供参考）。

ATX 版本

规范 说明
ATX 2.x 传统规范；12V 单路或多路
ATX 3.0 / 3.1 强调 12VHPWR / 12V-2x6 显卡供电、峰值负载；配 RTX 40 / 50 系高端卡建议优先

Cybenetics 等噪声评级可作补充参考，非强制。

命名与模组类型

示例：Seasonic FOCUS GX-850 ATX3

字段 含义
850 额定功率 850 W（看额定，非峰值）
GX / RMx 等 品牌内部系列
ATX3 支持 ATX 3.0 规范与相应显卡接口

模组类型

类型 说明 理线
非模组 线材固定 机箱较乱，便宜
半模组 主板 24pin + CPU 固定，其余可拆 常见折中
全模组 全部线材可拆 整洁，价略高

外形规格

规格 场景
ATX 电源 标准中塔机箱
SFX / SFX-L ITX 小机箱，见 机箱选购：规格、风道与兼容性
TFX 品牌机/OEM 窄机箱，DIY 少见

供电接口

接口 用途
24pin ATX 主板主供电
4+4pin EPS CPU 供电；高端板 / 高功耗 U 需 双 8pin
PCIe 6+2pin 显卡供电；8pin = 150 W 量级
12VHPWR / 12V-2x6 RTX 40 / 50 高端卡 单口 600 W 级；ATX 3.0 电源原生或转接
SATA / Molex 硬盘、风扇、灯带

显卡功耗越高，所需 8pin 数量 越多；买前对照显卡说明书与电源自带线材。转接线 尽量用电源原厂或显卡盒内线，避免劣质并联。

选购要点

  1. 功率留余量：按 CPU + GPU 峰值估算，主流游戏 650–750 W 金牌全模组 最常见。
  2. 优先金牌 + 靠谱型号：避免无评测的极端低价「炸弹电源」。
  3. ATX 3.0：新配 RTX 5070 Ti 及以上建议原生 12V-2x6 或认证转接方案。
  4. 线材与接口：核对显卡 pin 数、CPU 8pin 数量、SATA 硬盘数量。
  5. 保修：常见 5–10 年；海韵、振华、海盗船高端型号保修较长。
  6. 尺寸：超长电源（>160 mm）需确认 机箱选购：规格、风道与兼容性 电源仓长度。
