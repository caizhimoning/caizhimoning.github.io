Obsidian 社区插件与本库选用

发表于2026-06-09
|浏览量:

本库在核心插件之外另装一批社区插件，覆盖网页剪藏、设置检索、增强搜索、最近文件与 PDF 阅读批注。安装路径：设置 → 社区插件 → 浏览；用法总览见 Obsidian 学习与用法手册，WSL 环境见 Obsidian 设置：字体、外观与 Properties。

插件 类型 一句话
Obsidian Web Clipper 浏览器扩展 从网页剪藏到库

PDF.notes 社区插件 PDF 上画笔涂抹，批注写入文件

Obsidian Web Clipper

  • 类型：浏览器扩展（Chrome / Firefox 等），不是 Obsidian 内社区插件。
  • 作用：在网页上选中内容或整页，保存为 Obsidian 笔记（可指定库、文件夹、模板）。
  • 安装：Obsidian Web Clipper 官网 或各浏览器扩展商店。
  • 注意：剪藏目标库若在 WSL（/home/moning/obsidian_dir），需按官方说明配置与 Obsidian URI；WSLg 下剪贴板与 Linux 应用互通有限，见 [[obsidian-setting#4. 截图与剪贴板（补充）]]。

Settings Search

  • 社区名：Settings Search（作者 javalent）。
  • 作用：在 设置 界面顶部提供搜索框，按关键词跳转到对应选项（含部分第三方插件注册项）；支持方向键与 Enter 导航。
  • 场景：选项藏在多层菜单里时，比手动翻页快（如「附件路径」「模板文件夹」）。
  • 用法：打开设置后直接输入关键词即可，无需额外命令。

Recent Files

  • 社区名：Recent Files。
  • 作用：在侧栏显示最近打开过的笔记列表，点击即可回到上下文。
  • 场景：在 notes/ 多篇笔记、附件 PDF 之间来回切换时，比 Ctrl+O 快切换更贴近「刚才在看什么」。
  • 与核心插件关系：不替代快速切换（Ctrl+O）和书签；三者可并存。

Omnisearch

  • 社区名：Omnisearch。
  • 作用：BM25 相关度排序的搜索引擎，可视作 Quick Switcher 增强版：按文件名、标题、正文找笔记；配合 Text Extractor 可索引 PDF、Office、图片内文字。
  • 典型能力：引号精确匹配、-排除、按扩展名过滤（如 .md）、从结果插入 [[链接]]、Vault 内搜索与单文件内搜索切换。
  • 与核心「全局搜索」区别：全局搜索（Ctrl+Shift+F）偏全文列出匹配行；Omnisearch 偏快速定位文件并打开，交互更像 Ctrl+O。
  • 默认快捷键：以插件设置为准（常见为 Ctrl+Shift+O 或覆盖 Quick Switcher，安装后可在 设置 → Omnisearch 查看）。

PDF++

  • 社区名：PDF++（GitHub）。
  • 作用：增强内置 PDF 阅读器——划词高亮、复制选区链接、从反向链接显示高亮、工具栏颜色板等；高亮可与 Obsidian 双链结合（链到 PDF 选区即形成批注）。
  • 批注存储：可不修改 PDF 文件（链在笔记里），也可选择写入 PDF 标准批注层（对外部阅读器可见，有一定限制）。
  • 适合：教材、论文 精读、引文摘录；本库 PDF 多在 notes/attachments/（如力学教材），用 PDF++ 划词并 [[链接]] 回写笔记。
  • 不适合：自由 画笔涂抹（见 PDF.notes）。

PDF.notes

  • 社区名：PDF.notes（GitHub）。
  • 作用：在 Obsidian 内打开 PDF，用 画笔、形状、橡皮 在页面上直接标注；批注 自动写入 PDF 文件，可 flatten 成普通 PDF 查看器也能看到的墨迹。
  • 打开方式：打开 PDF 后点功能区画笔图标，或右键文件 → Open with PDF.notes。
  • 适合：习题册、讲义 手写圈画；与 PDF++ 划词高亮可分工配合。
  • 注意：社区插件商店若未收录，可用 BRAT 按 GitHub 地址安装；大 PDF 手写层可能增大文件体积。

安装与启用

  1. 设置 → 社区插件 → 关闭安全模式（若尚未开启社区插件）。
  2. 浏览 → 搜索上表英文名 → 安装 → 启用。
  3. Web Clipper 在浏览器单独安装，并在扩展选项里指向本库路径。

与本库核心插件的分工

需求 优先用
浏览文件夹 核心 文件列表
按标题跳笔记 核心 快速切换 或 Omnisearch
全文搜一句话 核心 全局搜索 或 Omnisearch
刚看过的几篇 Recent Files
PDF 划词 + 双链 PDF++
PDF 画笔涂抹 PDF.notes
找设置项 Settings Search
网页进库 Web Clipper
