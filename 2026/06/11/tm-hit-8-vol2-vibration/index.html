第四章 机械振动基础

机械振动指系统在其平衡位置附近的往复运动。本章以弹簧-质量系统为模型，建立固有频率、阻尼、受迫振动与共振概念，并推广到转子与二自由度系统。

从牛顿定律到振动方程（初学推导）

弹簧-质量系统，坐标 $x$ 以平衡位置为原点，弹簧力 $F_s = -kx$（恢复力指向平衡位置）：

$$m\ddot{x} = -kx \quad \Rightarrow \quad m\ddot{x} + kx = 0$$

这是二阶线性常微分方程。物理含义：加速度与位移成正比且反向 → 简谐运动。

§4-1 单自由度系统的自由振动

运动微分方程

质量 $m$，弹簧刚度 $k$，坐标 $x$ 自平衡位置起算：

$$m\ddot{x} + kx = 0$$

固有圆频率与周期

$$\omega_n = \sqrt{\frac{k}{m}}, \quad f_n = \frac{\omega_n}{2\pi}, \quad T = \frac{2\pi}{\omega_n} = 2\pi\sqrt{\frac{m}{k}}$$

$\omega_n$：固有圆频率；$f_n$：固有频率（Hz）；$T$：周期。

通解

$$x = A\sin(\omega_n t + \theta)$$

$A, \theta$ 由初始条件 $x_0, v_0$ 确定：

$$A = \sqrt{x_0^2 + \left(\frac{v_0}{\omega_n}\right)^2}$$

等效弹簧刚度

多个弹簧组合或复杂结构，可化为等效 $k_{\mathrm{eq}}$，仍用 $\omega_n = \sqrt{k_{\mathrm{eq}}/m}$。

组合 $k_{\mathrm{eq}}$
并联 $k_1 + k_2$
串联 $\dfrac{k_1 k_2}{k_1 + k_2}$

例题：求固有频率

例 1：$m=2,\text{kg}$，$k=200,\text{N/m}$ → $\omega_n=\sqrt{200/2}=10,\text{rad/s}$，$T=2\pi/10\approx 0.628,\text{s}$。

例 2：摆长 $l=1,\text{m}$ 的单摆（小角度）→ $\omega_n=\sqrt{g/l}=\sqrt{9.8}\approx 3.13,\text{rad/s}$。

例 3：弹簧并联 $k_1=100,,k_2=200$ → $k_{\mathrm{eq}}=300,\text{N/m}$。

初始条件

$x(0)=x_0$，$\dot{x}(0)=v_0$ → $A=\sqrt{x_0^2+(v_0/\omega_n)^2}$，$\tan\theta=-\omega_n x_0/v_0$（依符号象限调整）。

§4-2 计算固有频率的能量法

无阻尼自由振动中机械能守恒：$T + V = E = \text{常数}$。

设最大角位移处 $V_{\max} = T_{\max}$（通过平衡位置时动能最大）：

$$\omega_n = \sqrt{\frac{k}{m}} \quad \text{或能量法：} \quad V_{\max} = T_{\max}$$

能量法步骤：

  1. 设最大位移（振幅）为 $A$，写出 $V_{\max}$
  2. 过平衡位置时 $V=0$，$T_{\max}=\dfrac{1}{2}m v_{\max}^2$
  3. 令 $V_{\max}=T_{\max}$，结合运动学关系（如 $v_{\max}=\omega_n A$）解 $\omega_n$

例：均质细杆长 $l$、质量 $m$，下端铰支、上端连弹簧刚度 $k$，用势能极大与动能极大相等求 $\omega_n$，避免列微分方程。

§4-3 单自由度系统的有阻尼自由振动

粘性阻尼模型

阻尼力 $F_d = -c\dot{x}$，$c$ 为阻尼系数：

$$m\ddot{x} + c\dot{x} + kx = 0$$

阻尼比

$$\zeta = \frac{c}{2\sqrt{mk}} = \frac{c}{c_{\mathrm{cr}}}, \quad c_{\mathrm{cr}} = 2\sqrt{mk}$$

三种运动

条件 名称 解的特征
$\zeta < 1$ 欠阻尼 衰减振荡 $x = Ae^{-\zeta\omega_n t}\sin(\omega_d t + \theta)$，$\omega_d = \omega_n\sqrt{1-\zeta^2}$
$\zeta = 1$ 临界阻尼 最快无振荡回归平衡
$\zeta > 1$ 过阻尼 缓慢回归，无振荡

工程中小阻尼 $\zeta \ll 1$ 时，$\omega_d \approx \omega_n$。

§4-4 单自由度系统的无阻尼受迫振动

方程与稳态解

$$m\ddot{x} + kx = F_0\sin\omega t$$

稳态振幅：

$$A = \frac{F_0/k}{|1 - (\omega/\omega_n)^2|} = \frac{F_0}{|k - m\omega^2|}$$

共振

$\omega \to \omega_n$ 时 $A \to \infty$（无阻尼理想情形）。共振频率 $\omega_r = \omega_n$。

相位：$\omega < \omega_n$ 同相；$\omega > \omega_n$ 反相。

共振的物理直觉

干扰频率接近固有频率时，每一周期外力都在「推」系统往同一方向走，能量不断积累，振幅变大。类比：荡秋千时每次在恰当时候推一把。

区域 $\omega$ vs $\omega_n$ 振幅 相位
低频 $\omega \ll \omega_n$ $A\approx F_0/k$（准静态） 同相
共振 $\omega \approx \omega_n$ 极大 90°
高频 $\omega \gg \omega_n$ $A$ 很小 反相

§4-5 单自由度系统的有阻尼受迫振动

$$m\ddot{x} + c\dot{x} + kx = F_0\sin\omega t$$

稳态振幅：

$$A = \frac{F_0}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}}$$

共振时振幅有限，取决于 $\zeta$。动力放大系数 $\beta = A/A_{\mathrm{st}}$，$A_{\mathrm{st}} = F_0/k$ 为静变形。

§4-6 转子的临界转速

转子质量偏心 $e$，角速度 $\omega$，弹性轴刚度 $k$。横向振动方程类似受迫振动，当 $\omega$ 接近轴系横向固有频率 $\omega_n$ 时发生共振，对应转速为临界转速：

$$\omega_{\mathrm{cr}} \approx \omega_n = \sqrt{\frac{k}{m}}$$

设计转子工作转速应避开临界转速（亚临界或超临界运行）。

§4-7 隔振

主动隔振

机器（质量 $m$）经弹簧 $k$、阻尼 $c$ 安装在基础上，减小传到基础的力。

力传递率：

$$T_r = \frac{\sqrt{1 + (2\zeta\omega/\omega_n)^2}}{|1 - (\omega/\omega_n)^2|}$$

$\omega \gg \omega_n$ 时 $T_r < 1$，隔振有效。

被动隔振

减小基础振动传到仪器；形式类似，设计使仪器固有频率远低于干扰频率。

§4-8 两个自由度系统的自由振动

方程矩阵形式

$$\mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{0}$$

设 $\mathbf{x} = \mathbf{A}\sin(\omega t + \theta)$，得特征值问题：

$$(\mathbf{K} - \omega^2 \mathbf{M})\mathbf{A} = \mathbf{0}$$

非零解条件 $\det(\mathbf{K} - \omega^2 \mathbf{M}) = 0$ 给出两个固有频率 $\omega_1, \omega_2$；对应振型（模态）$\mathbf{A}^{(1)}, \mathbf{A}^{(2)}$。

主振动

每个固有频率对应一种主振动模式；一般自由振动为两种主振动的叠加。

§4-9 两个自由度系统的受迫振动 · 动力减振器

动力减振器（吸振器）

主系统 $m_1$-$k_1$ 上附加辅助质量 $m_2$ 与弹簧 $k_2$。调节 $m_2, k_2$ 使在干扰频率 $\omega$ 处主系统振幅为零（$\omega$ 等于辅助子系统固有频率）。

条件（无阻尼理想情形）：

$$\omega = \sqrt{\frac{k_2}{m_2}}$$

主系统 $m_1$ 在 $\omega$ 处振幅为零，振动能量转移到减振器。

二自由度例题思路

两质量 $m_1,m_2$，弹簧 $k_1,k_2,k_{12}$ 连接，列：

$$\begin{bmatrix} m_1 & 0 \ 0 & m_2 \end{bmatrix} \begin{bmatrix}\ddot{x}1 \ \ddot{x}2\end{bmatrix} + \begin{bmatrix} k_1+k{12} & -k{12} \ -k_{12} & k_2+k_{12} \end{bmatrix} \begin{bmatrix}x_1 \ x_2\end{bmatrix} = \mathbf{0}$$

设 $x_i = A_i\sin(\omega t)$，特征方程 $\det(\mathbf{K}-\omega^2\mathbf{M})=0$ 得 $\omega_1,\omega_2$。

本章公式速查

量 公式
固有圆频率 $\omega_n = \sqrt{k/m}$
周期 $T = 2\pi/\omega_n$
阻尼比 $\zeta = c/(2\sqrt{mk})$
欠阻尼振动 $x = Ae^{-\zeta\omega_n t}\sin(\omega_d t+\theta)$
共振（无阻尼） $\omega = \omega_n$ 时 $A\to\infty$

本章小结

  • 能由 $m,k$ 或能量法求 $\omega_n$
  • 能判断欠阻尼 / 临界 / 过阻尼
  • 受迫振动：分母 $|k-m\omega^2|$ 在 $\omega=\omega_n$ 最小 → 共振
  • 隔振：$\omega \gg \omega_n$ 时传递率 $<1$
