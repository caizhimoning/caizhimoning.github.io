LaTeX 语法入门

发表于2026-06-05
|浏览量:

LaTeX 是基于 TeX 的排版系统，通过命令与宏包描述文档结构与样式，由引擎编译为 PDF。适用于论文、报告、幻灯片与书籍。本文介绍最常用的 语法结构 与命令；具体宏包与编译方式因发行版（TeX Live、MiKTeX）而异。

文档骨架

最小可编译示例：

部分 作用
\documentclass[选项]{类名} 定义版式、字号、单双面等
\usepackage{宏包} 扩展功能（数学、图表、超链接等）
\begin{document} … \end{document} 正文区间

常见文档类：

类名 用途
article 论文、短文
report 报告、章节带章
book 书籍
beamer 幻灯片

章节与目录

命令 级别
\part 部（book / report）
\section 节
\subsection 小节
\subsubsection 小小节

article 无 \chapter；report / book 可用 \chapter{标题}。

文字与段落

段落与换行

空一行表示新段落。强制换行可用 \\ 或 \newline（不推荐滥用）。

字体与强调

命令 效果
\textbf{粗体} 粗体
\textit{斜体} 斜体
\texttt{等宽} 等宽（代码）
\emph{强调} 语义强调（通常斜体）

常用写法

效果 代码
分数 \frac{a}{b}
上下标 x^{2}, a_{i}
根号 \sqrt{x}, \sqrt[n]{x}
求和 / 积分 \sum_{i=1}^{n}, \int_{a}^{b}
希腊字母 \alpha, \beta, \Delta
括号自适应 \left( \frac{1}{2} \right)

图片与表格

选项 含义

表格

{lcc} 表示左对齐、居中、居中列。专业表格常用 booktabs 宏包（\toprule、\midrule、\bottomrule）。

标签与引用

须先 \label{键}，再 \ref / \eqref；编译通常需 两次 才能解析引用。

常用数学符号

类别 示例
关系 \leq, \geq, \neq, \approx
集合 \in, \subset, \cup, \cap, \emptyset
逻辑 \forall, \exists, \land, \lor
箭头 \rightarrow, \Rightarrow, \leftrightarrow

XeLaTeX Unicode 与系统字体友好；中文推荐
LuaLaTeX 与 XeLaTeX 类似，宏包 luatex 特性

与 Markdown 的关系

对比 Markdown LaTeX
定位 轻量标记、博客与笔记 精确排版、学术出版
公式 部分平台支持 $...$ 原生强项
学习曲线 低 较高

博客 Markdown 语法见 Markdown 语法入门；Obsidian 笔记可用 Markdown，需投稿期刊或学位论文时再使用 LaTeX。
