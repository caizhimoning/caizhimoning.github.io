机箱选购：规格、风道与兼容性

发表于2026-06-05
|浏览量:

机箱决定 主板与显卡尺寸上限、风道与扩展位，并影响理线与噪音。须与 主板选购：芯片组、命名与平台搭配 板型、显卡选购：品牌、命名与性能分级 显卡长度、电源选购：功率、认证与模组 电源规格及 散热器选购：风冷、水冷与机箱风扇 散热器高度匹配。下文介绍主要品牌、规格命名、风道类型及 2025–2026 年大陆电商常见价位（仅供参考）。

同价位下差异主要在 钢板厚度、防尘网、风扇位、理线空间、前面板接口。

机箱规格：ITX / M-ATX / ATX

机箱按支持的 最大主板板型 分级（向下兼容更小主板）：

机箱类型 支持主板 体积 典型场景
Full Tower 全塔 E-ATX / ATX 最大 多硬盘、分体水冷、工作站
Mid Tower 中塔 ATX / mATX / ITX 主流 大多数 DIY 游戏主机
Micro Tower / mATX 机箱 mATX / ITX 较小 省空间、预算装机
Mini-ITX 机箱 ITX only 最小 便携、客厅 HTPC

中塔 ATX 为 2025 年最常见选择：兼容标准 ATX 板、三风扇显卡与主流塔式散热。

关键兼容参数

购买前对照产品页 规格表，与已选配件核对：

参数 说明 关联配件
主板支持 ATX / mATX / ITX 主板选购：芯片组、命名与平台搭配
显卡限长 如 ≤ 380 mm 显卡选购：品牌、命名与性能分级 实测长度 + 余量
CPU 散热器限高 如 ≤ 165 mm 散热器选购：风冷、水冷与机箱风扇
电源限长 如 ≤ 200 mm 电源选购：功率、认证与模组；顶置电源仓更敏感
电源规格 ATX / SFX ITX 需 SFX 仓
3.5” / 2.5” 盘位 机械 / SATA SSD 数量 存储选购：固态、机械硬盘与接口
前置 I/O USB-A / Type-C、音频 按需
显卡竖装 PCIe 延长线槽位 可选，影响风道

显卡「几槽」厚度 影响下方风扇位与竖装；超长超厚卡避免窄机箱。

风道与风扇位

常见风道方案

类型 进风 出风 特点
前进后出 前部 后部 + 顶部 经典，兼容性好
下进上出 底部（电源仓分离） 顶部 海景房、O11 类
侧进 侧面网孔 后 / 顶 加强 GPU 进风

原则：进风略大于出风 形成微正压，减少灰尘从缝隙吸入；防尘网需定期清理。

风扇规格

尺寸 说明
120 mm 最普遍
140 mm 同转速风量更大、噪音略低
180 / 200 mm 部分全塔前面板

机箱 不含风扇 或仅配 1–2 把的情况常见；需另购风扇，见 散热器选购：风冷、水冷与机箱风扇。

玻璃侧透

钢化玻璃侧板利于展示 RGB，但 隔热略差；高功耗主机注意进风，避免玻璃贴死无风道。

性能分级与用途

档位 典型特征 适合
入门 钢板薄、风扇位少、无 Type-C 办公核显、低功耗
主流 中塔 ATX、显卡 ≤ 350 mm、前网孔 单卡游戏、标准塔散
散热加强 全网孔、多风扇位、理线仓 高功耗 CPU + 中高端显卡
紧凑 ITX NR200、A4 类；SFX 电源 小桌面、携带
展示 / 海景房 双面玻璃、仓室分区 外观优先，注意温度监控

大致价格区间

大陆电商常见区间（元，约 2025 下–2026 初，不含风扇）：

档位 举例 大致价位
入门 mATX / ATX 先马、爱国者入门 ¥100–200
主流 mesh 中塔 酷冷 MB、追风者 G 系 ¥250–450
主流海景房 / 仓室 乔思伯 D31、联力 LANCOOL ¥350–600
高端 airflow 海盗船 4000D、联力 Lancool III ¥500–800
全塔 / 静音旗舰 Fractal Define、联力 O11 高配 ¥700–1,200+
ITX 精品 NR200、Formd T1 类 ¥400–900+

预装 3–4 把风扇 的套装比裸箱贵 ¥80–200。

选购要点

  1. 先列兼容性清单：主板板型、显卡长厚、散热高度、电源长度与规格，四项同时满足。
  2. 优先网孔前面板：高功耗配置避免全封闭玻璃进风；mesh 版通常比纯玻璃版更凉。
  3. 理线空间：背板后宽度 ≥ 20–25 mm 更易走线；电源仓旁 SSD 位看 存储选购：固态、机械硬盘与接口 数量。
  4. 防尘与拆洗：顶部与底部滤网可拆；正压风道减灰。
  5. 运输与放置：玻璃侧板拆卸包装；勿长期堵死进风口（地毯、墙角）。
  6. 与散热联动：机箱风扇数量与 CPU 散热器搭配见 散热器选购：风冷、水冷与机箱风扇。
