内存选购：DDR4 / DDR5、品牌与频率

发表于2026-06-05
|浏览量:

内存（RAM）为 CPU 与程序提供高速工作空间；容量与频率直接影响多任务、游戏与创作流畅度。须与 主板选购：芯片组、命名与平台搭配 支持的 DDR4 或 DDR5 一致，并与 CPU 选购：品牌、命名与性能分级 平台匹配。下文介绍主要品牌、命名规则、性能分级与 2025–2026 年大陆电商常见价位（仅供参考）。

主要品牌

消费级台式机内存常见品牌：

品牌 系列（举例） 概括
金士顿 Kingston Fury Beast / Renegade 渠道广，兼容性好
芝奇 G.SKILL Trident Z / Ripjaws 高频与灯条选择多
海盗船 Corsair Vengeance / Dominator 中高端与 RGB 常见
英睿达 Crucial Pro / Ballistix 系（存量） 原厂颗粒，性价比
光威、阿斯加特、金百达 等 国产渠道牌 价格敏感、国产颗粒方案多
威刚 ADATA XPG Lancer 等 主流 DIY

原厂颗粒厂商还包括三星、海力士、美光；零售条多为 模组厂 封装。同规格下品牌差异常在散热、RGB、质保与兼容性验证。

DDR4 与 DDR5

项目 DDR4 DDR5
适用平台 Intel 10–14 代部分板、AM4 AM5、Intel 12 代起新板、LGA1851
插槽 288 pin，缺口与 DDR5 不通用 288 pin，不同缺口
常见频率 3200–3600 MHz 5600–6400 MHz（XMP/EXPO 更高）
电压 约 1.2 V 约 1.1 V（颗粒进化后高频条略高）
新装机建议 存量升级或极预算 2025 年起新平台首选

主板与 CPU 决定内存代数，不可混用。详见 主板选购：芯片组、命名与平台搭配。

命名与参数

示例：DDR5 6000 32GB (16GB×2) CL30

参数 含义
DDR4 / DDR5 代数
6000 等效频率 MT/s（习惯称 MHz）
32GB (16×2) 总容量；双通道 常见为两根套装
CL30 CAS 延迟（时序首项）；同频率下 CL 越小延迟越低
1.35 V 电压；超频条可能高于 JEDEC 标准

完整时序（进阶）

标注如 30-36-36-76 表示 CL-tRCD-tRP-tRAS。评测与超频可参考，日常选购 频率 + CL 即可。

一键超频配置

技术 平台
Intel XMP Intel 主板一键内存超频配置
AMD EXPO AMD 平台一键配置

购买时确认支持 XMP 或 EXPO（部分条两者兼有），并在 BIOS 中开启，否则可能仅跑在 JEDEC 默认频率（如 DDR5 4800）。

马甲与 RGB

  • 无马甲：入门条，办公够用。
  • 散热马甲：高频条常见，利于稳定。
  • RGB：外观向，对性能无实质提升。

性能分级与用途

用途 推荐容量 DDR5 频率参考 DDR4 频率参考
办公 / 网课 16 GB 5600 即可 3200
游戏主流 32 GB 6000–6400 3600
3A + 直播 / 多开 32–64 GB 6000+ 3600
视频剪辑 / 虚拟机 64 GB+ 6000+（容量优先） 32–64 GB
重度工作站 96–192 GB 依主板 QVL 与 CPU 较少新装

AMD Ryzen 对内存频率较敏感，常见甜点 DDR5 6000 CL30（EXPO）；Intel 13 / 14 / 15 代亦普遍适配 6000–7200，过高需看主板与 CPU 内存控制器。

游戏在 32 GB 双通道 已足够绝大多数场景；16 GB 在部分新 3A 中偏紧。

大致价格区间

套装价格（元，约 2025 下–2026 初）：

规格 大致价位 备注
DDR4 16GB (8×2) 3200 ¥180–280 存量平台升级
DDR4 32GB (16×2) 3600 ¥350–500
DDR5 16GB (8×2) 5600 ¥280–400 入门新平台
DDR5 32GB (16×2) 6000 CL30 ¥550–800 主流甜点
DDR5 32GB (16×2) 6400–7200 ¥700–1,000 高频溢价

国产颗粒品牌在同规格下常低 10%–25%；需接受不同批次兼容性验证差异。

选购要点

  1. 先定主板代数：DDR4 或 DDR5 不可混用；与 主板选购：芯片组、命名与平台搭配 一致。
  2. 优先双通道套装：买 2 根同规格套装，勿混不同品牌 / 频率 / 容量。
  3. 容量优先于极限频率：办公 16 GB；游戏 32 GB；创作考虑 64 GB+。
  4. 查 QVL（可选）：主板官网内存支持列表，可减少兼容问题（尤其 4 根满插高频时）。
  5. BIOS 开 XMP / EXPO：装系统后进 BIOS 开启，否则频率不达标。
  6. 安装插槽：双通道插法见 内存条安装：插槽位置与双通道规则；装机全流程见 DIY 装机流程：从配件到点亮。
  7. 散热与电源：高频条 + 塔扇需注意马甲高度与内存间距；整机供电见 电源选购：功率、认证与模组。
