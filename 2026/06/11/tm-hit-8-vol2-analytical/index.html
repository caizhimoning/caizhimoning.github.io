第一章 分析力学基础

在矢量力学方法中需要事先对系统中各质点的受力情况进行分析，而对于复杂约束系统，由于约束力的性质和分布在求解前是未知的，使得求解过程变得极为复杂，也无法建立一般力学系统的动力学方程。采用数学分析的方法来求解力学问题的理论在18世纪得到了迅速的发展，形成了“分析力学”的理论体系。

分析力学采用能量与功来描述物体运动与相互作用之间的关系，通过达朗贝尔原理和虚位移原理建立了普遍形式下的动力学方程，为现代动力学理论的发展奠定了基础，也对近代数学和物理学的发展起了巨大的推动作用。

分析力学以功和能量以及广义坐标为核心，统一处理完整约束下的平衡与运动。与第（Ⅰ）册几何静力学、矢量力学相比，方程形式更便于处理多自由度系统。

§1-1 自由度和广义坐标

位形与约束

$n$ 个质点组成的质点系，每个质点 3 个坐标，共 $3n$ 个坐标。若有 $s$ 个独立完整约束：

$$f_j(x_1, y_1, z_1, \ldots, x_n, y_n, z_n, t) = 0, \quad j = 1, \ldots, s$$

则自由度（独立运动参数个数）：

$$N = 3n - s$$

广义坐标

  • 描述质点系在空间中位置的独立参数，称为广义坐标
    当n个质点的系统受s个完整双侧约束:
  • 完整：约束方程只含坐标（不含速度），或可积分成这种形式
  • 双侧：约束是等式 f=0 ，不是不等式（如 f≥0 ）
    $$f_k(\mathbf{r}_1, \mathbf{r}_2, \cdots, \mathbf{r}_n, t) = 0 \quad (k=1,2,\cdots,s)$$

用 $N$ 个独立参数 $q_1, q_2, \ldots, q_N$ 确定系统位形：（独立参数是没有约束关系，相互独立）
（实际上就是左边是xyz，3n个参数，右边是3n-s+t，定常约束时没有t）

$$\mathbf{r}_i = \mathbf{r}_i(q_1, \ldots, q_N, t)$$

$q_k$ 称为广义坐标；可以是长度、角度等，不必具有长度量纲，数量为自由度数。

广义虚位移

[[变分（Variation）详解]]

  • 等时：变分时时间不变，δt=0
  • 变分：对函数形式做微小改变，类似微分但概念不同

$$\delta \mathbf{r}i = \sum{k=1}^{N} \frac{\partial \mathbf{r}_i}{\partial q_k} \delta q_k$$

$\delta q_k$ 为广义坐标 $q_k$ 的虚位移。

$$\delta \mathbf{r}i = \sum{k=1}^{N} \frac{\partial \mathbf{r}_i}{\partial q_k} \delta q_k \quad (i=1,2,\cdots,n) $$

符号 含义
$\delta \mathbf{r}_i$ 第 $i$ 个质点的虚位移（直角坐标系中的假想微小位移）
$N$ 系统的自由度数目，$N = 3n - s$
$\mathbf{r}_i$ 第 $i$ 个质点的位置矢量
$q_k$ 第 $k$ 个广义坐标
$\delta q_k$ 第 $k$ 个广义坐标的变分（广义虚位移）
$\frac{\partial \mathbf{r}_i}{\partial q_k}$ 位置对广义坐标的偏导数（变换系数）
$n$ 系统中质点的总数

物理意义

每个质点的虚位移，等于各广义坐标虚位移引起的位移的线性叠加。

换句话说：改变广义坐标 $q_k$ 会引起实际位置 $\mathbf{r}_i$ 的变化，把所有这些变化加起来，就是总的虚位移。

推导来源

由式 (1-4)：

$$\mathbf{r}_i = \mathbf{r}_i(q_1, q_2, \cdots, q_N, t)$$

对两边进行等时变分（$\delta t = 0$，时间冻结）：

$$\delta \mathbf{r}_i = \frac{\partial \mathbf{r}_i}{\partial q_1}\delta q_1 + \frac{\partial \mathbf{r}_i}{\partial q_2}\delta q_2 + \cdots + \frac{\partial \mathbf{r}_i}{\partial q_N}\delta q_N + \underbrace{\frac{\partial \mathbf{r}i}{\partial t}\delta t}{=0}$$

用求和符号写成：

$$\delta \mathbf{r}i = \sum{k=1}^{N} \frac{\partial \mathbf{r}_i}{\partial q_k} \delta q_k$$

与全微分的类比

全微分 $d\mathbf{r}_i$ 等时变分 $\delta \mathbf{r}_i$
定义 真实发生的无穷小变化 假想的、约束允许的无穷小变化
时间 $dt \neq 0$ $\delta t = 0$
公式 $d\mathbf{r}_i = \sum \frac{\partial \mathbf{r}_i}{\partial q_k}dq_k + \frac{\partial \mathbf{r}_i}{\partial t}dt$ $\delta \mathbf{r}_i = \sum \frac{\partial \mathbf{r}_i}{\partial q_k}\delta q_k$
用途 描述真实运动 用于虚功原理、推导拉格朗日方程

核心要点

  1. 虚位移 $\delta \mathbf{r}_i$ 不是真实发生的位移，而是满足约束条件的”假想”位移
  2. 广义虚位移 $\delta q_k$ 互相独立（因为 $q_k$ 是独立参数）
  3. 这个公式是虚功原理和拉格朗日方程推导的关键一步
  4. 偏导数 $\frac{\partial \mathbf{r}_i}{\partial q_k}$ 只与系统的几何约束有关，与运动状态无关

简单例子

一个质点在 $xy$ 平面上运动，选极坐标为广义坐标：$q_1 = r$，$q_2 = \theta$

位置矢量：
$$\mathbf{r} = r\cos\theta ,\mathbf{i} + r\sin\theta ,\mathbf{j}$$

则虚位移：
$$\delta \mathbf{r} = \frac{\partial \mathbf{r}}{\partial r}\delta r + \frac{\partial \mathbf{r}}{\partial \theta}\delta \theta = (\cos\theta ,\mathbf{i} + \sin\theta ,\mathbf{j})\delta r + (-r\sin\theta ,\mathbf{i} + r\cos\theta ,\mathbf{j})\delta \theta$$

其中：

  • $\frac{\partial \mathbf{r}}{\partial r}$ 是径向单位矢量方向
  • $\frac{\partial \mathbf{r}}{\partial \theta}$ 是横向单位矢量方向（大小为 $r$）

示例

系统 自由度 广义坐标
单摆（绳长 $l$） 1 摆角 $\varphi$
平面双摆 2 $\varphi_1, \varphi_2$
刚体平面运动 3 $x_C, y_C, \varphi$
刚体绕定点运动 3 欧拉角 $\psi, \theta, \varphi$

§1-2 以广义坐标表示的质点系平衡条件

广义力

与广义坐标 $q_k$ 对应的广义力：

$$Q_k = \sum_{i=1}^{n} \mathbf{F}i \cdot \frac{\partial \mathbf{r}i}{\partial q_k}
= \sum{i=1}^{n} \left( F{ix} \frac{\partial x_i}{\partial q_k} + F_{iy} \frac{\partial y_i}{\partial q_k} + F_{iz} \frac{\partial z_i}{\partial q_k} \right) \quad (k = 1, 2, \cdots, N)$$

质点系的平衡条件

虚功方程
$$\delta W_F = \sum_{k=1}^{N} Q_k \delta q_k = 0$$

具有理想约束的质点系在平衡位置，广义坐标具有独立性， $\delta q_k$可以取任意值， 有 $\delta W = 0$，得：

$$Q_k = 0, \quad k = 1, 2, \ldots, N$$

即各广义力为零——与 [[tm-hit-8-vol1-dynamics#§14-3 广义坐标 · 广义力]] 虚位移原理的坐标形式一致。

§1-3 保守系统的平衡条件 · 平衡稳定性

平衡条件

势力场中，$Q_k = -\partial V / \partial q_k$，平衡条件：

$$\frac{\partial V}{\partial q_k} = 0, \quad k = 1, \ldots, N$$

平衡位形为势能 $V(q_1,\ldots,q_N)$ 的驻点。

平衡稳定性

对单自由度系统，平衡位置 $q_0$ 处：

条件 性质
$\dfrac{\mathrm{d}^2 V}{\mathrm{d}q^2}\big _{q_0} > 0$
$\dfrac{\mathrm{d}^2 V}{\mathrm{d}q^2}\big _{q_0} < 0$
$\dfrac{\mathrm{d}^2 V}{\mathrm{d}q^2}\big _{q_0} = 0$

多自由度：势能矩阵 $\left( \dfrac{\partial^2 V}{\partial q_i \partial q_j} \right)$ 在平衡点正定 → 稳定。

例题：单摆平衡与稳定性

单摆势能 $V = mgl(1 - \cos\varphi)$（最低点 $\varphi=0$ 为零势能）。

  • $\dfrac{\mathrm{d}V}{\mathrm{d}\varphi} = mgl\sin\varphi = 0 \Rightarrow \varphi = 0, \pi$
  • $\dfrac{\mathrm{d}^2 V}{\mathrm{d}\varphi^2} = mgl\cos\varphi$
    • $\varphi=0$：$>0$ → 稳定（下垂平衡）
    • $\varphi=\pi$：$<0$ → 不稳定（倒立平衡）

§1-4 动力学普遍方程

推导逻辑（把两章拼起来）

达朗伯-拉格朗日原理

对具有理想约束的质点系，在任意虚位移上：

$$\sum_{i=1}^{n} (\mathbf{F}i + \mathbf{F}{Ii}) \cdot \delta \mathbf{r}_i = 0$$

$\mathbf{F}i$ 为主动力，$\mathbf{F}{Ii} = -m_i \mathbf{a}_i$ 为惯性力。
约束力不出现在方程中——因为理想约束下约束力虚功之和为零（与静力学虚位移原理相同）。

代入 $\delta \mathbf{r}_i = \sum_k \dfrac{\partial \mathbf{r}_i}{\partial q_k} \delta q_k$，由 $\delta q_k$ 独立，得动力学普遍方程：

$$\sum_{k=1}^{N} (Q_k + Q_{Ik}) \delta q_k = 0$$

其中 $Q_{Ik} = \sum_i \mathbf{F}_{Ii} \cdot \dfrac{\partial \mathbf{r}_i}{\partial q_k}$ 为惯性力对应的广义力。

等价形式（用动能 $T$）：

$$\sum_{k=1}^{N} \left( Q_k + \sum_{i=1}^{n} m_i \mathbf{a}_i \cdot \frac{\partial \mathbf{r}_i}{\partial q_k} \right) \delta q_k = 0$$

§1-5 第二类拉格朗日方程

对完整约束系统，经变换可得标准形式。记广义速度 $\dot{q}_k$，第二类拉格朗日方程：

$$\frac{\mathrm{d}}{\mathrm{d}t}\left( \frac{\partial T}{\partial \dot{q}_k} \right) - \frac{\partial T}{\partial q_k} = Q_k, \quad k = 1, \ldots, N$$

$T$ 为系统总动能。

有势力情形

若主动力均为有势力，引入拉格朗日函数：

$$L = T - V$$

则

$$\frac{\mathrm{d}}{\mathrm{d}t}\left( \frac{\partial L}{\partial \dot{q}_k} \right) - \frac{\partial L}{\partial q_k} = 0$$

完整例题：单摆的拉格朗日方程

已知：质量 $m$，摆长 $l$，重力 $g$，广义坐标 $\varphi$（与铅垂夹角）。

Step 1 写动能、势能

质点速度 $v = l\dot{\varphi}$：

$$T = \frac{1}{2}mv^2 = \frac{1}{2}ml^2\dot{\varphi}^2$$

取最低点为零势能：$V = mgl(1 - \cos\varphi)$。

Step 2 拉格朗日函数

$$L = T - V = \frac{1}{2}ml^2\dot{\varphi}^2 - mgl(1-\cos\varphi)$$

Step 3 代入方程 $\dfrac{\mathrm{d}}{\mathrm{d}t}\dfrac{\partial L}{\partial \dot{\varphi}} - \dfrac{\partial L}{\partial \varphi} = 0$

  • $\dfrac{\partial L}{\partial \dot{\varphi}} = ml^2\dot{\varphi}$ → $\dfrac{\mathrm{d}}{\mathrm{d}t}(\cdots) = ml^2\ddot{\varphi}$
  • $\dfrac{\partial L}{\partial \varphi} = -mgl\sin\varphi$

得：

$$ml^2\ddot{\varphi} + mgl\sin\varphi = 0 \quad \Leftrightarrow \quad \ddot{\varphi} + \frac{g}{l}\sin\varphi = 0$$

Step 4 小角度线性化 $\sin\varphi \approx \varphi$：

$$\ddot{\varphi} + \frac{g}{l}\varphi = 0 \quad \Rightarrow \quad \omega_n = \sqrt{g/l}$$

与 [[tm-hit-8-vol2-vibration#§4-1 单自由度系统的自由振动]] 单摆周期 $T=2\pi\sqrt{l/g}$ 一致。

列方程的一般流程（初学必记）

有非有势力时

若存在摩擦力等，方程为：

$$\frac{\mathrm{d}}{\mathrm{d}t}\frac{\partial L}{\partial \dot{q}_k} - \frac{\partial L}{\partial q_k} = Q_k^{(\text{非有势})}$$

例如粘性阻尼 $Q = -c\dot{q}$。

§1-6 拉格朗日方程的初积分

循环坐标（可 ignorable）

若 $L$ 不显含某 $q_k$，而 $\dot{q}_k$ 显含，则该坐标为循环坐标，有广义动量守恒：

$$p_k = \frac{\partial L}{\partial \dot{q}_k} = \text{常数}$$

广义能量积分

若 $L$ 不显含时间 $t$（定常约束、主动力有势），则广义能量守恒：

$$H = \sum_k \frac{\partial L}{\partial \dot{q}_k} \dot{q}_k - L = T + V = E = \text{常数}$$

（对常见力学系统，$H$ 即机械能。）

例题：中心力场中的角动量守恒

质点 $m$ 在平面内运动，极坐标 $(r,\theta)$，势能 $V(r)$ 仅与 $r$ 有关。

  • 广义坐标：$q_1=r,,q_2=\theta$
  • $L$ 不显含 $\theta$ → $\theta$ 为循环坐标
  • 守恒量 $p_\theta = \dfrac{\partial L}{\partial \dot{\theta}} = mr^2\dot{\theta} = \text{常数}$

即角动量守恒 $L_z = mr^2\dot{\theta}$——比直接用 $\mathbf{M}_O=\mathbf{0}$ 更系统。

何时用初积分？

现象 初积分 条件
某坐标不出现 $p_k=\text{常数}$ $L$ 不显含 $q_k$
能量守恒 $T+V=E$ $L$ 不显含 $t$，且只有有势力
对称性 动量 / 角动量守恒 Noether 定理（进阶）

§1-7 第一类拉格朗日方程

对约束 $\Phi_j = 0$，引入拉格朗日乘子 $\lambda_j$，第一类拉格朗日方程：

$$\frac{\mathrm{d}}{\mathrm{d}t}\left( \frac{\partial T}{\partial \dot{q}_k} \right) - \frac{\partial T}{\partial q_k} = Q_k + \sum_j \lambda_j \frac{\partial \Phi_j}{\partial q_k}$$

连同约束方程 $\Phi_j = 0$ 联立求解。乘子 $\lambda_j$ 即约束力沿约束法向的广义分量。
实际计算中，第二类方程（消去约束力）更常用；第一类用于需要求约束力时。

对比 第二类 第一类
未知量 $q_k$ $q_k$ + 乘子 $\lambda_j$
约束力 自动消去 由 $\lambda_j$ 求出
适用 求运动 求约束力（如绳张力）

本章小结（初学自检）

  • 能说出：$N = 3n - s$（完整约束），广义坐标是什么
  • 能区分：虚位移 $\delta \mathbf{r}$ vs 实位移 $\mathrm{d}\mathbf{r}$（$\delta t=0$）
  • 会对单摆写 $L$ 并推出运动方程
  • 知道循环坐标 → 广义动量守恒；$L$ 不显含 $t$ → 机械能守恒
  • 能复述：达朗伯原理 + 虚位移原理 → 动力学普遍方程 → 拉格朗日方程

与第（Ⅰ）册对照

第（Ⅰ）册 第（Ⅱ）册升级
虚位移原理 $\delta W_F=0$（静平衡） 加入惯性力虚功 → 动力学
对质点列 $m\mathbf{a}=\mathbf{F}$ 对系统列 $N$ 条关于 $q_k$ 的方程，不含约束力
直角坐标 $x,y$ 广义坐标 $q_k$（角度、弧长等更自然）
