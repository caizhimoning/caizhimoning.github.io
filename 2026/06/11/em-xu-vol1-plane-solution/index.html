第三章 平面问题的直角坐标解答

§3-1 多项式解答

（同李刚书 [[em-li-2021-plane#§3-7 多项式逆解法]]。）

阶次选择经验：

荷载特征 建议 $\phi$ 阶次
均匀拉压 二次
线性弯矩（纯弯） 二次
端部集中力（悬臂） 三次
均布荷载 四次
楔体自重 三次或四次

§3-2 端部受集中力的悬臂梁

$$\sigma_x = -\frac{P}{I}y\left(x - \frac{l}{2}\right), \quad \tau_{xy} = -\frac{P}{2I}\left(\frac{h^2}{4} - y^2\right)$$

§3-3 受均布荷载的简支梁

弯矩-曲率关系与材料力学一致；弹性力学给出横向正应力 $\sigma_y$ 的修正（通常较小）。

横向正应力（近似）：

$$\sigma_y \approx -\frac{q_0}{2}\left(1 - \frac{y^2}{h^2/4}\right) - \frac{q_0 y^2}{2h^2}\left(1 - \frac{y^2}{h^2/4}\right)$$

量级约为 $\sigma_y \sim q_0$；与 $\sigma_x \sim q_0 (l/h)^2$ 相比通常可忽略，但短梁 $l/h$ 较小时需考虑。

§3-4 楔形体受重力和液体压力

用三次、四次应力函数；楔角 $\alpha$ 与荷载决定应力场。

§3-5 半无限平面体在边界上受法向集中力

Boussinesq 平面问题（厚度方向平面应变）：表面集中力 $P$ 的应力场——为空间 Boussinesq 解的平面类比。

§3-6 孔口附近的应力集中

小圆孔 $r=a$，远场 $\sigma_x = S$：孔边最大切向应力 $3S$（[[em-li-2021-plane#§4-5 含小圆孔的无限大板受拉]]）。

第四章 平面问题的极坐标解答

§4-1 轴对称应力与位移

$\phi = \phi(r)$，$\tau_{r\theta} = 0$。

§4-2 厚壁圆筒受均匀压力

Lamé 公式（[[em-li-2021-plane#§4-3 圆环或圆筒（Lamé 解）]]）。

§4-3 曲梁的纯弯曲

中性轴不在几何中心；径向应力不可忽略。

§4-4 圆孔的应力集中

Kirsch 解；应力集中系数、孔边切向应力分布。

§4-5 楔形体在楔顶或楔面受力

集中力 $P$ 作用于楔顶，应力 $\sigma \sim 1/r$ 奇异性（理想化点力）。

§4-6 半无限楔或半无限平面

边界受分布荷载的积分叠加。

第五章 平面问题的复变函数解答

§5-1 应力函数与复变函数表示

复变量

令 $z = x + iy$，$\bar{z} = x - iy$。Airy 应力函数 $\phi(x,y)$ 为实函数，可表为解析函数组合的实部。

Kolosov-Muskhelishvili 表示的导出思路

定义组合应力

$$\Theta = \sigma_x + \sigma_y = \nabla^2 \phi, \quad \Xi = \sigma_y - \sigma_x + 2i\tau_{xy}$$

由 $\nabla^4\phi=0$ 知 $\Theta$ 为调和函数，可表为解析函数 $\varphi’(z)$ 的实部：$\Theta = 4\operatorname{Re}\varphi’(z)$。进一步可证 $\Xi$ 与 $z$、$\bar{z}$ 及 $\varphi,\psi$ 的导数满足

$$\sigma_x + \sigma_y = 4\operatorname{Re}\varphi’(z)$$

$$\sigma_y - \sigma_x + 2i\tau_{xy} = 2\left[\bar{z}\varphi’’(z) + \psi’(z)\right]$$

$\varphi(z),\psi(z)$ 为解析函数（复位势）。一旦求得 $\varphi,\psi$，应力分量由上述两式解出。复变法的优势：解析函数、共形映射可系统处理孔、裂纹等多连通域。

§5-2 边界条件的复变形式

给定边界力 $\bar{f}_x + i\bar{f}_y$：

$$\varphi(t) + t\overline{\varphi’(t)} + \overline{\psi(t)} = i\int (\bar{f}_x + i\bar{f}_y),\mathrm{d}s$$

$t$ 为边界点复坐标。

§5-3 有限域与无限域

  • 有限单连通域：$\varphi, \psi$ 单值；
  • 无限域含孔：$\varphi’(z), \psi’(z)$ 在无穷远趋于常数（远场应力给定）。

§5-4 圆孔含无限大板受拉

用 $z = \zeta + \frac{a^2}{\zeta}$ 将圆外映射到圆环，求 $\varphi, \psi$，得 Kirsch 解。

§5-5 椭圆孔

共形映射 $z = z(\zeta)$ 将椭圆外域映到圆外，求应力；椭圆孔边应力集中与长短轴比有关。

§5-6 裂纹尖端应力（Griffith）

裂纹尖端 $r \to 0$，应力 $\sigma \sim 1/\sqrt{r}$；应力强度因子 $K_I$ 表征奇异性强度，为断裂力学基础。

复变法解题流程（概要）

与应力函数法关系：复变表示是 Airy 函数的解析函数形式，便于处理孔、裂纹等多连通域。
