第六章 变质量动力学

火箭、喷气飞机、输送带落料等系统中，质点系质量随时间变化。不能把 $m$ 简单移入牛顿第二定律的常数，需建立变质量质点的运动方程。

为什么不能写 $m(t)\mathbf{a}=\mathbf{F}$？

对变质量系统，$\dfrac{\mathrm{d}}{\mathrm{d}t}(m\mathbf{v}) = m\dfrac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} + \dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}$，动量变化率包含质量变化项。密歇尔斯基方程正是把「喷出 / 并入质量的动量交换」整理成 $\dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}_r$ 项。

§6-1 变质量质点的运动微分方程

质量变化率

设质点质量 $m = m(t)$，$\dfrac{\mathrm{d}m}{\mathrm{d}t} < 0$ 表示喷射质量（系统质量减少）。

密歇尔斯基（Meshchersky）方程

变质量质点在主动力 $\mathbf{F}$ 作用下的运动微分方程：

$$m\frac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} = \mathbf{F} + \frac{\mathrm{d}m}{\mathrm{d}t} \mathbf{v}_r$$

符号 含义
$\mathbf{v}$ 质点绝对速度
$\mathbf{v}_r$ 相对速度：并入（或分离）质量相对主体的速度
$\dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}_r$ 反推力（推力）

$\dfrac{\mathrm{d}m}{\mathrm{d}t} > 0$ 表示吸收质量（如雨滴落入车厢），$\mathbf{v}_r$ 为被吸收部分相对主体的速度。

物理意义

动量定理对变质量系统：动量变化率 = 外力 + 由于质量流出入带来的动量交换。反推力与 $\mathbf{v}_r$ 方向相反（喷射时 $\mathbf{v}_r$ 为喷出物相对火箭速度，$\mathrm{d}m/\mathrm{d}t < 0$，反推力沿喷射反方向）。

齐奥尔科夫斯基火箭方程

火箭竖直向上，重力 $mg$，喷射相对火箭速度大小 $v_r$（常数），忽略空气阻力：

$$m\frac{\mathrm{d}v}{\mathrm{d}t} = -mg + v_r \left(-\frac{\mathrm{d}m}{\mathrm{d}t}\right)$$

令 $\mu = m_0/m$（质量比），积分得速度增量：

$$\Delta v = v_r \ln\frac{m_0}{m_f} - g t_{\mathrm{burn}}$$

结论：提高末速度需增大 $v_r$ 或质量比 $m_0/m_f$；多级火箭可进一步提高有效 $\Delta v$。

输送带落料例

车厢以速度 $v$ 水平匀速行驶，单位时间有质量 $\lambda$（kg/s）的物料水平速度为零地落入车厢。

  • 落入质量相对车厢：$\mathbf{v}_r$ 沿车运动反方向，大小 $v$
  • $\dfrac{\mathrm{d}m}{\mathrm{d}t} = \lambda > 0$（吸收质量）

水平方向：$0 = F + \lambda(-v)$ → 需牵引力 $F = \lambda v$ 维持匀速。

火箭方程积分推导（竖直上升，无阻力）

$$m\frac{\mathrm{d}v}{\mathrm{d}t} = -mg + v_r\left(-\frac{\mathrm{d}m}{\mathrm{d}t}\right)$$

设燃料以恒定相对速率 $v_r$ 喷出，$\dfrac{\mathrm{d}m}{\mathrm{d}t} = -\beta$（常数），分离变量：

$$\mathrm{d}v = -g,\mathrm{d}t + \frac{v_r}{m}(-\mathrm{d}m)$$

从 $t=0$（$v_0=0$，$m=m_0$）积分到 $t$（$m=m_f$）：

$$v = -gt + v_r\ln\frac{m_0}{m}$$

若燃烧时间 $t_b$，末速 $v = v_r\ln(m_0/m_f) - g t_b$。

数值例：$v_r=3000,\text{m/s}$，$m_0/m_f=5$，$t_b=100,\text{s}$：

$$v \approx 3000\times 1.609 - 980 \approx 3847,\text{m/s}$$（理想真空；实际大气阻力显著降低）

§6-2 变质量质点的动力学普遍定理

动量定理

$$\frac{\mathrm{d}(m\mathbf{v})}{\mathrm{d}t} = \mathbf{F} + \mathbf{F}_\Phi$$

$\mathbf{F}_\Phi = \dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}_r$ 为反推力。与 Meshchersky 方程等价。

动量矩定理

对定点 $O$：

$$\frac{\mathrm{d}\mathbf{L}_O}{\mathrm{d}t} = \mathbf{M}_O^{(F)} + \mathbf{M}_O^{(\Phi)}$$

需计入反推力对 $O$ 之矩。

动能定理

变质量系统动能变化：

$$\mathrm{d}T = \delta W_F + \delta W_\Phi + \text{质量交换项}$$

一般不能写为简单机械能守恒；喷射时化学能转化为动能。

与常质量系统的比较

项目 常质量 变质量
基本方程 $m\mathbf{a} = \mathbf{F}$ $m\dfrac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} = \mathbf{F} + \dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}_r$
动量 $m\mathbf{v}$ 守恒（无外力） 需含质量流项
应用 一般机械 火箭、喷气、落料输送

符号约定提醒

量 符号习惯
$\dfrac{\mathrm{d}m}{\mathrm{d}t}$ $>0$ 吸收质量；$<0$ 喷出质量
$\mathbf{v}_r$ 相对主体、**并入前（或分离前）**质量的速度
反推力 $\mathbf{F}_\Phi = \dfrac{\mathrm{d}m}{\mathrm{d}t}\mathbf{v}_r$，喷射时沿 $-\mathbf{v}_r$ 方向加速主体

本章小结

  • 会写 Meshchersky 方程，分清 $\mathbf{v}$ 与 $\mathbf{v}_r$
  • 会推火箭速度公式 $v = v_r\ln(m_0/m_f) - gt$
  • 落料 / 输沙问题：$\mathrm{d}m/\mathrm{d}t>0$，$\mathbf{v}_r$ 为相对速度
