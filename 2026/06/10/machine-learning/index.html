机器学习课程学习笔记

发表于2026-06-10
|浏览量:

机器学习（Machine Learning, ML） 研究如何让计算机从数据中自动学习规律，并对未见数据做出预测或决策。本笔记按常见本科/研究生《机器学习》课程脉络组织：先建立问题范式与数学工具，再讲经典算法、模型评估与正则化，最后概览神经网络与无监督学习。公式与符号与周志华《机器学习》（西瓜书）、Andrew Ng 课程及 sklearn 习惯对齐，便于对照教材与代码实践。

一、机器学习概述

1.1 定义

给定数据集 $D = {(x_1, y_1), \ldots, (x_m, y_m)}$，其中 $x_i$ 为样本（特征向量），$y_i$ 为标记或目标。机器学习的目标是学到一个函数 $f$，使 $f(x)$ 在未知样本上表现良好。

  • 特征（feature）：描述样本的属性，常记为 $d$ 维向量 $\mathbf{x} \in \mathbb{R}^d$。
  • 标记 / 标签（label）：监督学习中的目标 $y$；回归时 $y \in \mathbb{R}$，分类时 $y$ 为类别。
  • 假设空间（hypothesis space）：算法允许考虑的函数族 $\mathcal{H}$，如「所有线性函数」「深度为 $L$ 的神经网络」。

1.2 三类学习范式

范式 数据形式 典型任务 代表算法
监督学习 $(x, y)$ 成对 分类、回归 线性模型、SVM、树、神经网络
无监督学习 仅 $x$ 聚类、降维、密度估计 K-Means、PCA、GMM
强化学习 状态-动作-奖励序列 策略优化 Q-learning、策略梯度

本课程主体为监督学习与无监督学习；强化学习通常单开课程。

1.3 典型工作流

经验法则：测试集只在最终评估时使用一次；调参用验证集或交叉验证，避免「偷看测试集」导致乐观偏差。

二、课程知识地图

模块 核心问题 关键公式或概念
线性回归 连续 $y$ 如何拟合 最小二乘、梯度下降
分类 离散 $y$ 与概率 对数几率、交叉熵
评估 好坏如何度量 ACC、F1、AUC、MSE
正则化 如何抑制过拟合 L1/L2、早停
树与集成 非线性、可解释 信息增益、随机森林、GBDT
SVM 最大间隔分类 间隔、核技巧

无监督 结构发现 簇内距离、主成分

三、数学基础

3.1 线性代数（必备）

概念 在 ML 中的用途
向量、矩阵乘法 批量样本 $\mathbf{X} \in \mathbb{R}^{m \times d}$
转置、内积 线性模型 $\mathbf{w}^\top \mathbf{x} + b$
范数 $|\mathbf{w}|_2$、$|\mathbf{w}|_1$ L2/L1 正则化
特征值、特征向量 PCA、谱聚类
梯度 $\nabla f$ 梯度下降、反向传播

梯度下降核心：对可微损失 $L(\theta)$，迭代

$$\theta \leftarrow \theta - \eta \nabla_\theta L(\theta)$$

$\eta$ 为学习率；批量（全数据）、小批量（mini-batch）、随机（单样本）是三种常见变体。

3.2 概率论

概念 在 ML 中的用途
条件概率 $P(y \mid x)$ 分类器的概率输出
极大似然估计（MLE） 逻辑回归、高斯模型参数
贝叶斯公式 朴素贝叶斯分类器
期望、方差 偏差-方差分析
常见分布 高斯（回归噪声）、伯努利（二分类）、多项式（多分类）

MLE 思想：选参数使观测数据出现的概率最大。等价于最小化负对数似然，与交叉熵损失在分类问题中一致。

3.3 凸优化（了解）

  • 凸函数：局部极小即全局极小；线性回归（平方损失）为凸。
  • 非凸：神经网络损失一般非凸，依赖初始化与优化技巧（Adam、学习率调度）。
  • 拉格朗日乘子：约束优化；SVM 对偶推导的基础。

四、线性模型：回归

4.1 模型形式

单变量：$f(x) = wx + b$；多元：$f(\mathbf{x}) = \mathbf{w}^\top \mathbf{x} + b$。

给定训练集，均方误差（MSE）：

$$L(\mathbf{w}, b) = \frac{1}{m} \sum_{i=1}^{m} (f(\mathbf{x}_i) - y_i)^2$$

4.2 解析解（正规方程）

对线性回归，令设计矩阵 $\mathbf{X}$ 增广一列 1，可写 $\hat{\mathbf{w}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}$（在 $\mathbf{X}^\top \mathbf{X}$ 可逆时）。特征维数很高时求逆代价大，常用梯度下降。

4.3 梯度下降要点

要点 说明
学习率 $\eta$ 过大震荡，过小收敛慢；可试 $10^{-3} \sim 10^{-1}$ 量级
特征缩放 各维量纲差异大时，标准化 $(x - \mu)/\sigma$ 加速收敛
收敛判据 损失变化小于阈值或达最大迭代次数

4.4 概率视角

假设 $y = \mathbf{w}^\top \mathbf{x} + b + \epsilon$，$\epsilon \sim \mathcal{N}(0, \sigma^2)$，则 MLE 等价于最小化 MSE。

五、线性模型：分类

5.1 二分类：逻辑回归

直接回归 $[0,1]$ 概率不便，用对数几率（logit）：

$$P(y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x} + b)}}$$

$\sigma$ 为 sigmoid。损失为对数损失 / 二元交叉熵：

$$L = -\frac{1}{m}\sum_{i} \big[ y_i \log \hat{p}_i + (1-y_i)\log(1-\hat{p}_i) \big]$$

决策边界：$\mathbf{w}^\top \mathbf{x} + b = 0$，为线性分类器（在特征空间中是超平面）。

5.2 多分类：Softmax 回归

$$P(y=k \mid \mathbf{x}) = \frac{e^{\mathbf{w}k^\top \mathbf{x}}}{\sum{j} e^{\mathbf{w}_j^\top \mathbf{x}}}$$

损失为多类交叉熵。逻辑回归是多类 softmax 在 $K=2$ 时的特例。

5.3 与「感知机」的关系

  • 感知机：误分类则更新权重；在线算法；不输出概率。
  • 逻辑回归：概率模型 + 可微损失，便于梯度优化与校准。

六、泛化与模型选择

6.1 训练误差 vs 泛化误差

  • 训练误差：模型在训练集上的误差。
  • 泛化误差：模型在同分布新数据上的期望误差。

目标是最小化泛化误差，但只能间接通过验证策略估计。

6.2 过拟合与欠拟合

现象 训练误差 验证误差 对策
欠拟合 高 高 更复杂模型、更多特征、训练更久
过拟合 低 高 正则化、更多数据、简化模型、早停、集成

6.3 偏差-方差分解（回归）

对测试点 $x$，期望预测误差可分解为：

$$\mathbb{E}[(y - \hat{f}(x))^2] = \text{Bias}^2 + \text{Var} + \text{不可约噪声}$$

  • 偏差（Bias）：模型类太简单，系统性拟合不足。
  • 方差（Variance）：模型对训练集抽样过于敏感。

集成方法（Bagging 降方差、Boosting 降偏差）在此框架下理解更清楚。

七、模型评估

7.1 数据划分

集合 用途
训练集 拟合参数
验证集 选超参、选模型结构
测试集 最终无偏估计泛化性能

数据少时用 $k$ 折交叉验证：将训练部分划分为 $k$ 份，轮流作验证，其余训练，取 $k$ 次平均作为验证分数。

7.2 分类指标

设正类为「感兴趣类」（如患病、欺诈）：

指标 公式 / 含义
准确率 ACC $(TP+TN)/(TP+TN+FP+FN)$；类别不平衡时易误导
精确率 P $TP/(TP+FP)$ — 预测为正中有多少真为正
召回率 R $TP/(TP+FN)$ — 真实为正中有多少被找出
F1 $2PR/(P+R)$ — P 与 R 的调和平均
AUC ROC 曲线下面积；排序能力，与阈值无关

混淆矩阵是以上指标的共同基础。

7.3 回归指标

指标 公式
MSE $\frac{1}{m}\sum (y_i - \hat{y}_i)^2$
RMSE $\sqrt{\text{MSE}}$，与 $y$ 同量纲
MAE $\frac{1}{m}\sum |y_i - \hat{y}_i|$，对异常值更鲁棒
$R^2$ $1 - \frac{\sum(y_i-\hat{y}_i)^2}{\sum(y_i-\bar{y})^2}$

八、正则化

在损失中加入对复杂度的惩罚，抑制过拟合：

$$L_{\text{reg}} = L_{\text{data}} + \lambda \Omega(\theta)$$

类型 $\Omega$ 效果
L2（岭回归） $|\mathbf{w}|_2^2$ 权重整体变小，光滑解
L1（Lasso） $|\mathbf{w}|_1$ 稀疏解，特征选择
Elastic Net L1 + L2 组合 相关特征时比纯 L1 更稳

早停（early stopping）：验证误差不再下降时停止训练，等价于隐式正则化。

Dropout（神经网络）：训练时随机丢弃神经元，降低共适应，属结构正则化。

九、决策树与集成学习

9.1 决策树

用树结构做 if-then 规则划分特征空间：

  • 分类树：叶子节点为类别；分裂准则常用信息增益（ID3）、增益率（C4.5）、基尼指数（CART）。
  • 回归树：叶子为常数或局部均值；分裂准则常用均方误差减少量。

优点：可解释、可处理混合特征、非线性。缺点：单棵树易过拟合、不稳定。

信息熵（离散 $y$）：

$$H(D) = -\sum_{k} p_k \log_2 p_k$$

信息增益 = 父节点熵 − 加权子节点熵；偏好取值多的特征时，C4.5 用增益率修正。

9.2 Bagging 与随机森林

  • Bagging：自助采样（bootstrap）训练多个基学习器，投票 / 平均；降低方差。
  • 随机森林（RF）：Bagging + 每棵树随机子特征；强默认基线，抗过拟合。

9.3 Boosting

串行训练弱学习器，每轮关注前一轮残差或错例：

算法 特点
AdaBoost 调整样本权重
GBDT 梯度提升决策树；回归、分类常用
XGBoost / LightGBM 工程优化版 GBDT；竞赛与工业常见

Boosting 通常降低偏差，但需控制学习率与树深度以防过拟合。

十、支持向量机

10.1 最大间隔

二分类线性可分：找超平面 $\mathbf{w}^\top \mathbf{x} + b = 0$，使两类间隔最大。间隔与 $|\mathbf{w}|$ 相关，优化问题：

$$\min_{\mathbf{w},b} \frac{1}{2}|\mathbf{w}|^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^\top \mathbf{x}_i + b) \ge 1$$

10.2 软间隔与核技巧

  • 软间隔：引入松弛变量，允许少量误分类，参数 $C$ 控制间隔与误分类权衡。
  • 核函数 $K(\mathbf{x}_i, \mathbf{x}_j)$：隐式映射到高维，线性不可分时获得非线性边界。常见：线性核、多项式核、RBF（高斯）核。

$$K_{\text{RBF}}(\mathbf{x}, \mathbf{x}’) = \exp\left(-\frac{|\mathbf{x}-\mathbf{x}’|^2}{2\sigma^2}\right)$$

SVM 在高维稀疏文本分类、中小规模数据上仍常用；大规模深度学习中让位于神经网络。

十一、神经网络与深度学习入门

11.1 单层感知机 → 多层网络

多层前馈网络（MLP）：输入 → 隐藏层（非线性激活）→ 输出。

$$h^{(l)} = \sigma(W^{(l)} h^{(l-1)} + b^{(l)})$$

常见激活：ReLU $\max(0, x)$、sigmoid、tanh；输出层回归常用线性，分类用 softmax。

11.2 反向传播（Backpropagation）

利用链式法则，从输出层向输入层计算各参数梯度，再与梯度下降结合。计算图是自动微分（PyTorch、TensorFlow）的抽象基础。

11.3 训练技巧（课程常讲）

技巧 作用
小批量 SGD / Adam 稳定、高效优化
批归一化（BatchNorm） 稳定分布，加速训练
Dropout 正则化
学习率衰减 / warmup 后期精细收敛
数据增强 图像旋转、裁剪等，扩增有效样本

11.4 典型架构（拓展阅读）

架构 数据类型 代表任务
CNN 网格数据（图像） 分类、检测、分割
RNN / LSTM / Transformer 序列 NLP、时间序列
自编码器 / VAE 无监督表示 降维、生成
GAN / Diffusion 生成建模 图像生成

深度学习通常作为机器学习课程的后半延伸或独立课程。

十二、无监督学习

12.1 聚类

K-Means：划分 $k$ 个簇，最小化簇内平方和：

$$J = \sum_{i=1}^{k} \sum_{\mathbf{x} \in C_i} |\mathbf{x} - \boldsymbol{\mu}_i|^2$$

迭代：分配样本到最近中心 → 更新中心为簇均值。需预设 $k$；对初值和簇形状敏感。

其他：层次聚类、DBSCAN（密度聚类，任意形状）、GMM（软聚类 + EM）。

12.2 降维

主成分分析（PCA）：找方差最大的正交方向，将 $d$ 维投影到 $k$ 维（$k \ll d$）。步骤：中心化 → 协方差矩阵 → 特征分解 → 取前 $k$ 个特征向量。

用途：可视化、去噪、加速下游学习、缓解维度灾难。

t-SNE / UMAP：非线性降维，侧重可视化。

12.3 密度估计与生成（简介）

  • 高斯混合模型（GMM）：多个高斯分量 + EM 拟合。
  • 自编码器：编码-解码重构，学习低维表示。
