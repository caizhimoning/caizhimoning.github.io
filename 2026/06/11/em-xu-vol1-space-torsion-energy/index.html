第八章 空间问题的基本理论

§8-1 平衡微分方程

$$\sigma_{ij,j} + f_i = 0$$

§8-2 几何方程

$$\varepsilon_{ij} = \frac{1}{2}(u_{i,j} + u_{j,i})$$

§8-3 物理方程

广义胡克定律（[[em-li-2021-fundamentals#§2-6 广义胡克定律（本构方程）]]）。

§8-4 边界条件 · 圣维南原理

位移边、应力边；局部荷载用等效分布力代替。

§8-5 按位移求解 · Navier 方程

$$(\lambda+G)\nabla(\nabla\cdot\mathbf{u}) + G\nabla^2\mathbf{u} + \mathbf{f} = 0$$

§8-6 按应力求解 · 相容方程

Beltrami-Michell 方程（6 个相容关系）。

§8-7 轴对称问题

用 $r, \theta, z$，$u_r, u_\theta, u_z$；轴对称时 $u_\theta = 0$，$\partial/\partial\theta = 0$。

第十一章 能量原理与变分法

§11-3 最小势能原理

第十二章 弹性波的传播

§12-1 无限介质中的波动方程

运动方程

无体力时 $\sigma_{ji,j} = \rho \ddot{u}i$。代入本构 $\sigma{ij} = \lambda\delta_{ij}\theta + 2G\varepsilon_{ij}$ 与几何方程，得 Navier 运动方程

$$(\lambda + G)\nabla(\nabla\cdot\mathbf{u}) + G\nabla^2\mathbf{u} = \rho \ddot{\mathbf{u}}$$

纵波与横波的分解

令 $\mathbf{u} = \nabla\Phi + \nabla\times\mathbf{H}$（$\Phi$ 为标量势，$\mathbf{H}$ 为矢量势）。无源区可证 $\nabla^2\Phi - \dfrac{1}{c_p^2}\ddot{\Phi}=0$，$\nabla^2\mathbf{H} - \dfrac{1}{c_s^2}\ddot{\mathbf{H}}=0$，其中

$$c_p = \sqrt{\frac{\lambda + 2G}{\rho}}, \quad c_s = \sqrt{\frac{G}{\rho}}$$

$c_p$ 为纵波（膨胀波）速度，质点振动平行传播方向；$c_s$ 为横波（畸变波）速度，质点振动垂直传播方向。各向同性弹性体 $c_p > c_s$。

用 $E,\nu,\rho$ 表示：

$$c_p = \sqrt{\frac{E(1-\nu)}{\rho(1+\nu)(1-2\nu)}}, \quad c_s = \sqrt{\frac{E}{2\rho(1+\nu)}}$$

§12-2 平面波

位移 $u = f(x - ct)$ 沿 $x$ 传播；纵波、横波解耦。

§12-3 表面波（Rayleigh 波）

半无限弹性体自由表面传播的二维波，质点轨迹椭圆，能量集中于表层，波速 $c_R \approx 0.92 c_s$（泊松比 $\nu = 0.25$）。

§12-4 杆中的纵波

细长杆近似：$c = \sqrt{E/\rho}$（一维纵波速度）。

§12-5 反射与折射

波在界面反射、透射；Snell 定律类比。
