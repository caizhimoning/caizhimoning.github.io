DIY 装机流程：从配件到点亮

发表于2026-06-05
|浏览量:

DIY 装机是将已选配件按正确顺序组装为可开机系统的过程。配件选型见本系列各篇（CPU 选购：品牌、命名与性能分级、显卡选购：品牌、命名与性能分级、主板选购：芯片组、命名与平台搭配 等）；本文侧重 操作顺序、注意事项与首次点亮。内存插槽与双通道插法见专文 内存条安装：插槽位置与双通道规则。

装机前准备

工具与环境

物品 用途
十字螺丝刀（PH0 / PH1） 机箱、主板、电源、散热器
扎带 / 魔术贴 理线
防静电措施 触摸金属机箱释放静电；避免地毯摩擦；可选防静电手环
良好照明 看清针脚与跳线
宽敞台面 铺软垫或主板包装盒，避免刮伤背面焊点

核对清单

  • 所有配件型号兼容（CPU 插槽、内存 DDR 代数、机箱限长 / 限高、电源功率与接口）
  • 主板说明书、机箱说明书（纸质或 PDF）
  • 系统安装 U 盘（Windows / Linux）已制作
  • 显示器、键鼠接在 主板后置 I/O（无独显时）或 显卡接口（有独显时）

推荐安装顺序

两种常见流程均可行；下列为 先在桌面试装 CPU / 内存 / 散热器，再进机箱，便于排查问题：

水冷：冷排可在装主板前进机箱，或主板进箱后装；按机箱说明书决定顶部 / 前置，避免与内存、供电散热片干涉。

分步说明

1. 安装 CPU

  1. 打开主板 CPU 座拉杆（Intel 或 AMD 按箭头方向操作，勿暴力）。
  2. 对齐 三角标记 / 缺角，轻放入 CPU，勿按压针脚（AMD 针脚在 CPU 上；Intel LGA 针脚在主板）。
  3. 扣下拉杆。详见 CPU 选购：品牌、命名与性能分级 平台说明。

2. 安装内存

按 内存条安装：插槽位置与双通道规则 插入正确插槽，两侧卡扣 完全扣紧 直至听到咔嗒声。开机前再确认是否插到底。

3. 安装 M.2 固态

  1. 拧下主板 M.2 散热片螺丝（如有）。
  2. 固态 斜插入槽，压下后用螺丝固定。
  3. 装回散热片。接口与数量见 存储选购：固态、机械硬盘与接口、主板选购：芯片组、命名与平台搭配。

4. 安装 CPU 散热器

  • 风冷：涂硅脂（预涂则跳过）→ 对准扣具 → 对角拧紧。
  • 水冷：冷头固定于 CPU，冷排固定于机箱，风扇线接主板 CPU_FAN 或水泵口。

限高与扣具见 散热器选购：风冷、水冷与机箱风扇。

5. 主板进机箱

  1. 机箱内装好 铜柱，位置与主板孔位一致（ATX / mATX）。
  2. 安装 I/O 挡板（若未预装）。
  3. 主板对准挡板，拧螺丝 勿过紧，避免 PCB 弯曲。

6. 安装电源

  1. 电源风扇朝向：常见为 下置电源仓进风（底部有滤网时）。
  2. 固定螺丝拧紧。
  3. 预留 24pin 主板供电、CPU 8pin、显卡供电线，稍后插接。见 电源选购：功率、认证与模组。

7. 机箱前面板与风扇

线缆 常见接法
POWER SW 电源开关，无极性
RESET SW 重启
POWER LED / HDD LED 按说明书正负极
USB 3.0 / Type-C 对应主板针脚
HD AUDIO 前置耳机孔

机箱风扇 4pin PWM 接 SYS_FAN 等；CPU 风扇勿接错口。

8. 安装显卡

  1. 拆下机箱对应 PCIe 挡板。
  2. 打开主板插槽 卡扣，插入至卡扣自动扣合。
  3. 用螺丝固定挡板，接 PCIe 供电（如需）。见 显卡选购：品牌、命名与性能分级。

9. 接线汇总

必接 说明
24pin ATX 主板主供电
4+4pin CPU 主板左上角；高功耗 CPU 可能需双 8pin
显卡 6+2pin / 12V-2x6 按显卡要求
M.2 已插槽则无需额外线
SATA 2.5 / 3.5 硬盘

10. 理线与通电前检查

  • 无螺丝、扎带碎片掉落在主板上。
  • 所有供电线插到底。
  • 显卡、内存、散热器无松动。
  • 不要在通电时插拔内存或显卡。

首次开机与 BIOS

  1. 显示器接 显卡（有独显时）或 主板 HDMI/DP（核显）。
  2. 短接开机或按机箱电源键。
  3. 无显示：检查内存、显卡是否插牢、供电是否接全；听 蜂鸣码（如有喇叭）或看主板 Debug LED。
  4. 进 BIOS / UEFI（常见 Del / F2）：
    • 开启 XMP / EXPO（内存选购：DDR4 / DDR5、品牌与频率）
    • 确认识别到 M.2 与硬盘
    • 启动顺序选 U 盘安装系统
  5. 安装操作系统后安装 芯片组、显卡驱动。

常见错误

现象 可能原因
完全无反应 电源开关线接错、电源未开、插座无电
风扇转无画面 内存未插好、显示器接错口、显卡未供电
反复重启 内存不兼容、CPU 供电未插、短路（铜柱错位）
高温报警 散热器未撕膜、硅脂未涂、风扇线未接
识别不到硬盘 M.2 未插到底、BIOS 未开启对应模式

装机后检查

  • BIOS 中 CPU 温度、风扇转速正常
  • 所有硬盘容量正确
  • 双通道内存已识别（见 内存条安装：插槽位置与双通道规则）
  • 运行压力测试（可选）与 Windows 更新
  • 重要数据 备份策略（存储选购：固态、机械硬盘与接口）
