第一章 静力学公理和物体的受力分析

§1-1 静力学公理

基本概念

  • 力：物体间相互机械作用的可度量效应；作用于刚体时只改变运动状态。力是矢量，记 $\mathbf{F}$ 或 $(F_x, F_y, F_z)$。度量单位：N（牛）。
  • 刚体：变形可忽略，体内任意两点距离保持不变的物体。
  • 平衡：刚体相对惯性参考系静止或作匀速直线平动。
  • 力系：作用在刚体上的一组力。
  • 等效力系：对同一刚体产生相同运动效应的两个力系互换。
  • 合力 / 分力：与力系等效的一个力称合力；合力所代换的各力称分力。

力的表示

  • 作用线：力沿其作用方向的直线。
  • 作用点：力作用在刚体上的位置。
  • 力矢：有向线段表示，起点可沿作用线移动（对刚体等效）。

静力学公理（教材共 5 条）

公理 1（二力平衡公理）
刚体受两个力作用而平衡的必要充分条件是：两力等值、反向、共线。

公理 2（加减平衡力系公理）
在已知力系上加上或减去任意平衡力系，不改变原力系对刚体的等效作用。

公理 3（力的平行四边形定律）
作用于同一点的两力，其合力 $\mathbf{F}_R$ 以两力矢为邻边所作平行四边形的对角线表示，作用点仍在该点：

$$\mathbf{F}_R = \mathbf{F}_1 + \mathbf{F}_2$$

公理 4（共面汇交力系的平衡）
若各力作用线汇交于一点，则该汇交力系平衡的必要充分条件是合力为零（力多边形自行封闭）。

公理 5（刚化公理）
变形体在力系作用下处于平衡，若将变形体刚化，则平衡状态保持不变。

推论（由公理导出，非公理）

推论 1（力的可传性）
作用于刚体上某点的力，可沿其作用线移至刚体内任意另一点，不改变对刚体的效应。（由公理 1、2 导出）

推论 2（三力平衡汇交定理）
刚体受三个力作用而平衡，若其中两力的作用线汇交于一点，则第三力的作用线必过该汇交点。

推论 3（二力构件）
只受两个力作用的刚体平衡时，两力必等值、反向、共线；构件受力沿两端铰连线方向。

§1-2 约束和约束力

基本概念

  • 自由体：位移不受限制的物体。
  • 非自由体 / 约束体：位移受限制的物体。
  • 约束：限制物体某些位移的条件或物体。
  • 约束力（约束反力）：约束对物体的反作用，属于被动力。
  • 主动力：重力、推力、压力等，一般已知或可设。

常见约束及其约束力

约束 限制的运动 约束力特征
光滑面 沿公法线方向位移 沿接触处公法线，指向物体；记 $F_N$
光滑圆柱铰链（销钉） 相对转动 过铰心，方向未知；用 $F_x, F_y$（平面）或 $F_x, F_y, F_z$（空间）
光滑固定铰支座 平动 同铰链，分解为两个正交分力
滚动支座（活动铰） 法向位移 沿支承面法线，一个力
固定端 平动与转动 约束力 $\mathbf{F}$ 与约束力偶 $\mathbf{M}$
无重柔索 沿索伸长方向 沿索，拉力，不能受压
二力杆 — 力沿杆轴线

光滑铰链约束力的两种画法

平面铰链：可用一对正交分力 $F_x, F_y$，或沿杆件方向的力与另一未知力；最终计算结果与画法无关。

§1-3 物体的受力分析和受力图

受力分析步骤

  1. 确定研究对象（取分离体）。
  2. 分析作用在研究对象上的外力（主动力 + 约束力）。
  3. 画受力图：画出全部外力；内力不画。

画受力图注意

  • 只画研究对象所受的力，不画它对其他物体作用的力。
  • 同一作用力与反作用力分别画在各自研究对象上。
  • 二力构件受力必沿两铰连线。
  • 刚化同一铰链处多个构件时，铰链对构件的力才画在该构件上。

§1-4 力学模型与力学简图

  • 质点：几何尺寸可忽略，质量集中于一点。
  • 刚体：不计变形。
  • 力学简图：用理想约束（铰、光滑面、固定端等）代替实际连接，用集中力或分布力表示载荷。

第二章 平面力系

§2-1 平面汇交力系

基本概念

平面汇交力系：各力作用线位于同一平面内且汇交于一点。

合成

几何法：力多边形法则。依次作力三角形，合力 $\mathbf{F}_R$ 从起点指向终点。

解析法
建立直角坐标系 $Oxy$，各力投影：

$$F_{ix} = F_i \cos\theta_i, \quad F_{iy} = F_i \sin\theta_i$$

合力投影：

$$F_{Rx} = \sum F_{ix}, \quad F_{Ry} = \sum F_{iy}$$

合力大小与方向：

$$F_R = \sqrt{F_{Rx}^2 + F_{Ry}^2}, \quad \cos\theta = \frac{F_{Rx}}{F_R}, \quad \sin\theta = \frac{F_{Ry}}{F_R}$$

平衡

几何条件：力多边形自行封闭。

平衡方程（必要且充分）：

$$\sum F_x = 0, \quad \sum F_y = 0$$

独立方程数 2，最多求 2 个未知量。

投影式另一写法：

$$\sum F_i \cos\alpha_i = 0, \quad \sum F_i \sin\alpha_i = 0$$

其中 $\alpha_i$ 为力与 $x$ 轴夹角。

§2-2 平面力对点之矩 · 平面力偶

力对点之矩

定义：力 $\mathbf{F}$ 对点 $O$ 之矩为代数量 $M_O(F)$，等于力的大小与力臂 $h$ 的乘积：

$$M_O(F) = \pm F \cdot h$$

  • 力臂 $h$：点 $O$ 到力作用线的垂直距离。
  • 正负号：力使物体绕 $O$ 逆时针转动为正，顺时针为负。

解析式（力作用点 $A(x,y)$，力 $F_x, F_y$）：

$$M_O(F) = x F_y - y F_x$$

合力矩定理

平面力系中，合力对平面内任一点之矩，等于各分力对同一点之矩的代数和：

$$M_O(\mathbf{F}_R) = \sum M_O(\mathbf{F}_i)$$

力偶

力偶：等值、反向、不共线的两平行力 $\mathbf{F}$ 与 $\mathbf{F}’$ 组成的力系。

力偶矩 $M$（代数量）：

$$M = F \cdot d$$

$d$ 为两力作用线间距离（力偶臂）。

性质

  1. 力偶对刚体只产生转动效应，无平动效应。
  2. 力偶对任意点取力矩都等于力偶矩，不因矩心的改变而改变。。
  3. 只要保持力偶矩不变，力偶可在其作用面内任意移转，且可以同时改变力偶中力的大小与力偶臂的长短，对刚体的作用效果不变。
  4. 同向同平面力偶可合成：$M = \sum M_i$。

平面力偶系的合成与平衡

合成：$M = \sum M_i$

平衡的充要条件：所有各力偶矩对代数和等于零。
方程：

$$\sum M = 0$$

§2-3 平面任意力系的简化

力的平移定理

平面内作用在刚体上点 $A$ 的力 $\mathbf{F}$，可平移到另一点 $B$，但必须附加一力偶，其矩等于原力对 $B$ 点之矩：

$$M_B = M_B(\mathbf{F}) = F \cdot d$$

$d$ 为 $B$ 到原力作用线的距离。

向一点简化

平面任意力系向 $O$ 点简化，得：

  • 主矢 $\mathbf{F}_R’$：

$$F_{Rx}’ = \sum F_{ix}, \quad F_{Ry}’ = \sum F_{iy}, \quad F_R’ = \sqrt{F_{Rx}’^2 + F_{Ry}’^2}$$

主矢与简化中心无关。

  • 主矩 $M_O$：

$$M_O = \sum M_O(\mathbf{F}_i)$$

主矩一般与简化中心有关。

简化结果

| 情况 | 条件 | 最简力系 |
| — | ————————– | ————————- — ——– |
| 平衡 | $F_R’ = 0,\ M_O = 0$ | 零 |
| 合力偶 | $F_R’ = 0,\ M_O \neq 0$ | 力偶矩 $M = M_O$ |
| 合力 | $F_R’ \neq 0,\ M_O = 0$ | 合力过 $O$ |
| 合力 | $F_R’ \neq 0,\ M_O \neq 0$ | 合力，作用线距 $O$ 为 $d = \frac{ M_O }{F_R’}$ |

合力作用线位置：在主矩转向一侧，距 $O$ 点 $d$。

主矩的基点无关量

对任意两点 $O$、$O’$：

$$M_{O’} = M_O + M_{O’}(\mathbf{F}_R’)$$

当 $F_R’ \neq 0$ 时，存在使 $M_O = 0$ 的简化中心（合力作用线上）。

固定端约束力

固定端约束可简化为：约束力 $\mathbf{F}$ 与约束力偶 $M$。

§2-4 平面任意力系的平衡条件和平衡方程

平衡的必要充分条件

主矢为零且对任一点主矩为零：

$$F_R’ = 0, \quad M_O = 0$$

基本形式

$$\sum F_x = 0, \quad \sum F_y = 0, \quad \sum M_O(F_i) = 0$$

三个独立方程，可求三个未知量。

二矩式

$$\sum F_x = 0 \quad (\text{或 } \sum F_y = 0)$$

$$\sum M_A = 0, \quad \sum M_B = 0$$

限制条件：$A$、$B$ 连线不得垂直于投影轴（否则方程不独立）。

三矩式

$$\sum M_A = 0, \quad \sum M_B = 0, \quad \sum M_C = 0$$

限制条件：$A$、$B$、$C$ 三点不得共线。

平面平行力系

![[Snipaste_2026-06-09_17-52-34.png]]
各力作用线相互平行。独立平衡方程 2 个（若不限于投影）或 3 个中的两个独立形式：

$$\sum F_y = 0, \quad \sum M_O = 0$$

或

$$\sum F_y = 0, \quad \sum M_A = 0, \quad \sum M_B = 0$$

（$A$、$B$ 连线不与各力平行）

§2-5 物体系的平衡 · 静定和超静定问题

外力与内力

  • 外力：研究对象以外物体对其作用力（含支座约束力）。
  • 内力：研究对象内部各物体间相互作用力，成对出现。

物体系平衡方程

整个系统平衡时，任一部分均平衡。可对整体或任一子系统列平衡方程。

静定与超静定

  • 静定问题：全部未知约束力个数 = 独立平衡方程个数。
  • 超静定（静不定）问题：未知约束力个数 > 独立平衡方程个数，差值称为超静定次数。

§2-6 平面简单桁架的内力计算

桁架理想假设

  1. 杆均为直杆，自重不计。
  2. 节点为光滑铰链。
  3. 载荷作用于节点。
  4. 各杆轴线汇交于节点。

杆件内力

杆件对节点的作用力沿杆轴线，称杆内力。拉力为正，压力为负。

节点法

取节点为研究对象，列平面汇交力系平衡：

$$\sum F_x = 0, \quad \sum F_y = 0$$

截面法

用假想截面截断桁架，取其中一部分为研究对象，列平面任意力系平衡求指定杆内力。

零杆判断（三杆节点）

无外力作用的节点，若有两杆共线，则第三杆为零杆（内力为零）。

第三章 空间力系

§3-1 空间汇交力系

合成

$$\mathbf{F}_R = \sum \mathbf{F}_i$$

$$F_{Rx} = \sum F_{ix},\quad F_{Ry} = \sum F_{iy},\quad F_{Rz} = \sum F_{iz}$$

$$F_R = \sqrt{F_{Rx}^2 + F_{Ry}^2 + F_{Rz}^2}$$

平衡方程

$$\sum F_x = 0, \quad \sum F_y = 0, \quad \sum F_z = 0$$

§3-2 力对点的矩和力对轴的矩

力对点的矩（力矩矢量）

![[Snipaste_2026-06-10_14-42-46.png]]
力 $\mathbf{F}$ 作用点相对 $O$ 的位矢 $\mathbf{r}$，力对 $O$ 之矩：

$$\mathbf{M}_O(\mathbf{F}) = \mathbf{r} \times \mathbf{F}$$

大小：

$$|\mathbf{M}_O(\mathbf{F})| = F \cdot h = |\mathbf{r}| \cdot |\mathbf{F}| \cdot \sin\theta$$

方向：垂直于 $\mathbf{r}$ 与 $\mathbf{F}$ 构成的平面，按右手定则。

用行列式计算（$\mathbf{r}=(x,y,z)$，$\mathbf{F}=(F_x,F_y,F_z)$）：

$$\mathbf{M}_O(\mathbf{F}) = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \ x & y & z \ F_x & F_y & F_z \end{vmatrix}$$

合力矩定理

$$\mathbf{M}_O(\mathbf{F}_R) = \sum \mathbf{M}_O(\mathbf{F}_i)$$

力对轴的矩

力对坐标轴 $x, y, z$ 之矩等于力矩矢量在该轴上的投影：
$$M_x(\mathbf{F}) = \mathbf{M}O(\mathbf{F{xy}})=\pm F_{xy} h$$
按照右手定则，拇指与z轴一致时为z正
$$M_x(\mathbf{F}) = [\mathbf{M}_O(\mathbf{F})]_x = y F_z - z F_y$$

$$M_y(\mathbf{F}) = z F_x - x F_z$$

$$M_z(\mathbf{F}) = x F_y - y F_x$$

一般轴：力对任意轴 $l$ 之矩 = $\mathbf{M}_O(\mathbf{F})$ 在 $l$ 上的投影。

力与轴相交或平行：$M_l(\mathbf{F}) = 0$。

§3-3 空间力偶

力偶矩矢量

空间力偶矩 $\mathbf{M}$ 为矢量：

  • 大小 $|\mathbf{M}| = F d$
  • 方向垂直于力偶作用面，按右手定则

性质

  1. 力偶矩矢为自由矢量，空间力偶可以平移到与其作用面平行的任意平面上而不改变力偶对刚体的作用效果。
  2. 力偶矩矢与简化中心无关。
  3. 同向同空间力偶系合成：$\mathbf{M} = \sum \mathbf{M}_i$。

平衡

$$\sum \mathbf{M}i = 0 \quad \Leftrightarrow \quad \sum M{ix} = 0,\ \sum M_{iy} = 0,\ \sum M_{iz} = 0$$

§3-4 空间任意力系向一点简化

主矢与主矩

$$\mathbf{F}_R’ = \sum \mathbf{F}_i$$

$$\mathbf{M}_O = \sum \mathbf{M}_O(\mathbf{F}_i)$$

主矢与简化中心无关；主矩 $\mathbf{M}O$ 与简化中心有关。
求主矢大小方向:
$$F’R = \sqrt{\left(\sum F{ix}\right)^2 + \left(\sum F{iy}\right)^2 + \left(\sum F_{iz}\right)^2}
$$
$$\cos(F’R, \boldsymbol{i}) = \frac{\sum F{ix}}{F’_R}, \quad
\cos(F’R, \boldsymbol{j}) = \frac{\sum F{iy}}{F’_R}, \quad
\cos(F’R, \boldsymbol{k}) = \frac{\sum F{iz}}{F’_R}$$

求主矩大小方向:
$$M_O = \sqrt{\left(\sum M_{ix}\right)^2 + \left(\sum M_{iy}\right)^2 + \left(\sum M_{iz}\right)^2}$$

$$\cos(M_O, \boldsymbol{i}) = \frac{\sum M_{ix}}{M_O}, \quad
\cos(M_O, \boldsymbol{j}) = \frac{\sum M_{iy}}{M_O}, \quad
\cos(M_O, \boldsymbol{k}) = \frac{\sum M_{iz}}{M_O}$$
对 $O’$ 点：

$$\mathbf{M}_{O’} = \mathbf{M}O + \mathbf{M}{O’}(\mathbf{F}_R’) = \mathbf{M}O + \mathbf{r}{OO’} \times \mathbf{F}_R’$$

空间任意力系简化结果分析

情况 结果
$\mathbf{F}_R’ = 0,\ \mathbf{M}_O = 0$ 平衡
$\mathbf{F}_R’ = 0,\ \mathbf{M}_O \neq 0$ 合力偶
$\mathbf{F}_R’ \neq 0,\ \mathbf{M}_O = 0$ 合力
$\mathbf{F}_R’ \neq 0,\ \mathbf{M}_O \neq 0$，且 $\mathbf{F}_R’ \cdot \mathbf{M}_O = 0$ 合力
$\mathbf{F}_R’ \neq 0,\ \mathbf{M}_O \neq 0$，且 $\mathbf{F}_R’ \cdot \mathbf{M}_O \neq 0$ 力螺旋

力螺旋

合力 $\mathbf{F}_R’$ 与合力偶矩 $\mathbf{M}_O$ 不垂直时，最简力系为沿某轴线的一个力与一力偶，力偶矩矢量与力共线。其中力的作用线垂直于力偶的作用面

不变量：主矩在沿主矢方向的分量 $M_a = \mathbf{M}_O \cdot \mathbf{e}_R$（$\mathbf{e}_R$ 沿 $\mathbf{F}_R’$ 单位矢量）与简化中心无关。

§3-5 空间任意力系平衡方程

基本方程（六个独立方程）

$$\sum F_x = 0, \quad \sum F_y = 0, \quad \sum F_z = 0$$

$$\sum M_x = 0, \quad \sum M_y = 0, \quad \sum M_z = 0$$

空间任意力系平衡的必要充分条件：主矢 $\mathbf{F}_R’ = 0$ 且对任一点主矩 $\mathbf{M}_O = 0$。

空间平行力系

独立平衡方程 3 个，例如：

$$\sum F_z = 0, \quad \sum M_x = 0, \quad \sum M_y = 0$$

§3-6 重心

基本概念

  • 重力：地球对物体的吸引力。工程中常将物体各微元重力视为平行力系。
  • 重心：该平行力系的合力作用点，记 $C$。刚体上重心的位置与刚体在空间中的方位无关。
  • 形心：几何体的几何中心，由几何尺寸决定。
  • 质心：质量分布的中心。

均质刚体：重心、形心、质心重合。

重心的坐标公式（质点系 / 离散体）

各微元重量为 $P_i$，作用点坐标 $(x_i, y_i, z_i)$，总重 $P = \sum P_i$，重心坐标：

$$x_C = \frac{\sum P_i x_i}{P}, \quad y_C = \frac{\sum P_i y_i}{P}, \quad z_C = \frac{\sum P_i z_i}{P}$$

矢量形式：

$$\mathbf{r}_C = \frac{\sum P_i \mathbf{r}_i}{\sum P_i}$$

连续体的重心（积分形式）

微元重量 $dP = \gamma , dV$（$\gamma$ 为容重），总重 $P = \int_V \gamma , dV$：

$$x_C = \frac{1}{P} \int_V x , \gamma , dV, \quad y_C = \frac{1}{P} \int_V y , \gamma , dV, \quad z_C = \frac{1}{P} \int_V z , \gamma , dV$$

若用质量表示（$\rho$ 为密度，$m = \int_V \rho , dV$）：

$$x_C = \frac{1}{m} \int_V x \rho , dV$$

（$y_C$、$z_C$ 同理。）

均质物体的形心公式

$\gamma = \text{常数}$ 时，$P = \gamma V$，重心即形心：

$$x_C = \frac{1}{V} \int_V x , dV, \quad y_C = \frac{1}{V} \int_V y , dV, \quad z_C = \frac{1}{V} \int_V z , dV$$

均质等厚薄板（面积 $A$）：

$$x_C = \frac{1}{A} \int_A x , dA, \quad y_C = \frac{1}{A} \int_A y , dA$$

均质细棒（弧长 $L$）：

$$x_C = \frac{1}{L} \int_L x , dl, \quad y_C = \frac{1}{L} \int_L y , dl$$

对称性

若物体具有对称面、对称轴或对称中心，则重心必在对称面、对称轴或对称中心上。

组合体重心的求法

将复杂物体分成若干简单部分，各部分重量 $P_i$，重心坐标 $(x_i, y_i, z_i)$：

分割法：

$$x_C = \frac{\sum P_i x_i}{\sum P_i}, \quad y_C = \frac{\sum P_i y_i}{\sum P_i}, \quad z_C = \frac{\sum P_i z_i}{\sum P_i}$$

负重量法（挖空补偿法）
物体可看作完整基本体减去挖去部分。挖去部分重量取负值代入上式：

$$x_C = \frac{\sum P_i x_i}{\sum P_i}$$

其中挖去部分 $P_j < 0$。

均质组合体亦可用体积（或面积）代重量：

$$x_C = \frac{\sum V_i x_i}{\sum V_i}$$

（有挖空时 $V_j$ 取负。）

合力矩定理求重心

重力平行力系对某轴之矩等于各分重力对该轴之矩的代数和。对 $y$ 轴取矩：

$$P \cdot x_C = \sum P_i x_i \quad \Rightarrow \quad x_C = \frac{\sum P_i x_i}{P}$$

（对 $x$ 轴、$z$ 轴同理可得 $y_C$、$z_C$。）

常见均质简单图形的形心（平面）

图形 形心位置
矩形 对角线交点
三角形 各边中线的交点，距底边 $h/3$
半圆形（半径 $R$） 对称轴上，距直径 $\dfrac{4R}{3\pi}$
四分之一圆 各对称边界距边 $\dfrac{4R}{3\pi}$
扇形（圆心角 $\theta$，半径 $R$） 对称轴上，距圆心 $\dfrac{2R\sin(\theta/2)}{3\theta/2}$（$\theta$ 为弧度）

第四章 摩擦

§4-1 滑动摩擦

库仑摩擦定律

静滑动摩擦：摩擦力 $F_s$ 方向与相对滑动趋势相反，大小满足

$$0 \leq F_s \leq F_{s,\max}$$

最大静摩擦力：

$$F_{s,\max} = f_s N$$

$f_s$：静摩擦系数；$N$：法向约束力。

动滑动摩擦：滑动时

$$F_k = f_k N$$

$f_k$：动摩擦系数，一般 $f_k < f_s$。

§4-2 摩擦角和自锁现象

全约束力

法向约束力 $N$ 与摩擦力 $F$ 的合力 $\mathbf{F}_R$ 称全约束力。

摩擦角

静摩擦时全约束力与法线夹角 $\varphi$ 满足 $\tan\varphi = F/N$。

摩擦角 $\varphi_f$（临界状态）：

$$\tan \varphi_f = f_s$$

最大全约束力与法线夹角即为 $\varphi_f$。

自锁

若主动力合力 $\mathbf{F}_P$ 的作用线与法线夹角 $\theta \leq \varphi_f$，则无论力多大，物体保持相对静止——自锁。

斜面自锁则是斜面的倾角小于等于材料的摩擦角
自锁条件：

$$\theta \leq \varphi_f \quad \Leftrightarrow \quad \tan\theta \leq f_s$$

§4-3 考虑摩擦时物体的平衡问题

平衡时摩擦力为未知量，除平衡方程外还须：

$$F_s \leq f_s N$$

临界平衡：$F_s = f_s N$，可求使系统平衡或开始滑动的极值载荷。

解题时须先假设滑动趋势以定摩擦力方向，若结果 $F_s < 0$ 则方向假设相反。

§4-4 滚动摩阻的概念

滚动摩阻

物体滚动时，接触处有阻碍滚动的滚动摩阻力偶 $M$，其矩满足

$$0 \leq M \leq M_{\max}$$

$$M_{\max} = \delta N$$
实验表明:最大滚动摩阻力偶矩M与滚子半径无关，与支承面的正压力成正比
$\delta$：滚动摩阻系数（长度量纲，单位 m 或 mm）。
![[Snipaste_2026-06-10_19-46-44.png]]
$$F_{\text{滑}} = F_{\max} = f_s F_N = f_s P $$

与滑动摩擦比较

滚动时接触处无宏观滑动，一般 $\delta \ll f_s$（量级上），故滚动比滑动省力。
